## 3️⃣ Personel Yönetimi

### İzin Talebi
```
Personel → talep ekler → senin telefonuna bildirim
Sen → "İncele" → "Onayla" / "Reddet"
Reddedersen → SEBEBİ yaz (zorunlu)
```

### Fazla Mesai Onayı
**ÖNEMLİ:** 30 dk altı mesai SAYILMAZ (DOSPRESSO kuralı, DECISIONS#39).

```
Personel mesaiyi kapatamadıysa → otomatik mesai algılanır → senin onayını bekler
Sen → "Onayla" / "Reddet"
Onayladığın mesai bordroda yer alır
Onaylamadığın → SİLİNİR (personel maaşına eklenmez)
```

⚠️ **30 dk üstü mesai için Aslan'a sormaya gerek yok**, sen onaylayabilirsin. Ama hafta toplamı 11 saati geçerse Mahmut'a haber ver.

### PIN Değiştirme / Sıfırlama
- Personel PIN unutursa → Mahmut'a (HQ) WhatsApp at
- Mahmut PIN sıfırlar → personel `0000` ile girer → kendi PIN'ini belirler

---

## 4️⃣ Müşteri Şikayeti / Geri Bildirim

### Şikayet Geldi
```
1. Müşteriden duyunca → Telefonda "CRM" menüsü
2. "Yeni Şikayet" → ad/soyad/ürün/tarih
3. "Çözüm" alanını doldur (ne yaptın?)
4. Kaydet → Yavuz Coach'a otomatik bildirim
```

**Yetersiz kaldıysan:** Yavuz → Utku → Aslan zinciri.

### Müşteri Geri Bildirim (NPS - olumlu/olumsuz)
- Pilot süresince elimizde QR kod yok, MANUEL kayıt
- Telefondan "CRM" → "Geri Bildirim" → 5 yıldız + yorum

---

## 5️⃣ Stok ve Sipariş

### Stok Uyarı Aldım
- Telefondan "Stok" → kritik seviyedeki ürünleri gör
- Samet'e (HQ Satınalma) WhatsApp at + sistem üzerinden de "Sipariş Talebi"
- Samet onaylarsa → tedarikçiye iletilir

### Sipariş Geldi (depodan)
- Samet onayı sonrası tedarikçi ürün getirir
- Telefondan "Mal Kabul" → ürünleri gir + miktar + tarih
- Hata varsa "Mal Kabul İtiraz" → Samet'e geri gider
