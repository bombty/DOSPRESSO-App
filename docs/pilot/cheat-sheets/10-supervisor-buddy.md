# DOSPRESSO Cheat Sheet — Supervisor Buddy

**Hedef Kullanıcı**: Yönetim asistanı, supervisor'a destek (kariyer merdiveninin 4. seviyesi)
**Cihaz**: Şube tableti (kiosk) + kendi telefonun (Benim Günüm)
**Erişim**: Supervisor'ın çoğu yetkisi + ek yönetim eğitim modülleri

---

## 1. Supervisor Buddy Nedir?

Sen kariyer merdiveninin **4. seviyesindesin** (Stajyer → Bar Buddy → Barista → **Supervisor Buddy** → Supervisor).
Görevin: Supervisor'a yardım et, vardiya yönetimini öğren, ekip yönlendirmesi pratik yap, 7 yönetim modülünü bitir, **85 puan eşiğini aşıp Supervisor ol.**

**Önemli:** Bu seviyede personel atama, izin onay, kalite raporu yetkilerin var **ama** kritik kararlarda (mali, disiplin, müşteri şikayet eskalasyonu) supervisor onayı gerekir.

1. Tablet → "Şube Kiosk"
2. Adını listeden seç
3. PIN gir (4 haneli)
4. "Giriş" → Mission Control (Supervisor Buddy)

### Yöntem B — Kendi Telefonun
1. Tarayıcıda sistem aç
2. Kullanıcı adı: [size verilen, örn `sup_buddy_isiklar_1`]
3. Parola: SMS/WhatsApp
4. "Giriş Yap"

## 3. Ana Ekran

### Kioskta (Şubede)
| Bölüm | İçerik |
|---|---|
| **Vardiya Özet** | Mesai başlangıcı, ekipte kim var, açık görevler |
| **Görev Akışı** | Sürmekte / Beklemede / Tamamlandı |
| **Kalite Kontrol** | Bugün denetim listesi (checklist) |
| **Görev Havuzu** | Üstlen + ata (ikisi de yapabilirsin) |
| **Hızlı Aksiyonlar** | "Görev Ata", "Vardiya Notu", "Müşteri Geri Bildirim" |

### Telefonda (Benim Günüm)
| Bölüm | İçerik |
|---|---|
| **Eğitim İlerleme** | 7 yönetim modülü (liderlik, çatışma, KPI, koçluk, vb.) |
| **Supervisor Buddy → Supervisor İlerleme** | %0-100 bar, eşik 85 |
| **Mentörlük Notları** | Sana atanan stajyer/bar buddy değerlendirmeleri |

---

## 4. Günlük İş Akışı (3 Adım)

### Vardiya Başı (08:00 / 16:00)
1. Kiosk girişi → adın + PIN
2. Supervisor ile 5 dk brifing (sözlü) — bugünkü öncelik
3. Vardiya açılış checklist'i: "Kalite Kontrol" → bugünkü liste
4. Ekibe görev ata (supervisor'a danışarak)

### Vardiya Ortası
1. Görev kontrol: tamamlanmamışlara bakım
2. Stok hızlı sayım (atandıysa)
3. Yeni gelen müşteri şikayetlerini supervisor ile birlikte ele al
4. Bar buddy/stajyer eğitim noktalarını gözlemle

### Vardiya Sonu
1. Açık görevleri sonraki vardiyaya devret
2. Gün sonu kalite checklist kapat
3. Vardiya notu yaz (supervisor'a paralel)

---

## 5. Sık Kullanılan İşler

### Yeni Görev Atama (Supervisor ile Birlikte)
1. Mission Control → "+ Yeni Görev"
2. Atanan kişi seç (barista, stajyer, vb.)
3. Açıklama yaz (1-2 cümle)
4. Son tarih: bugün/yarın
5. **Pilot süresince:** Atamadan önce supervisor'a göster, onay al

### İzin Talebi Onayı
1. Sağ üst çan → "İzin Talebi"
2. Açıklamayı oku
3. **3 gün altı** → kendin onaylayabilirsin
4. **3 gün üstü** → supervisor'a yönlendir
5. Onay/Red sonra mudur'a otomatik bildirim

### Eğitim Modülü Çalışma (Telefondan)
1. Telefondan "Benim Günüm" → "Eğitimlerim"
2. 7 yönetim modülü sırayla
3. **Geçme notu: 85** (Supervisor Buddy seviyesi — yüksek)
4. Modül + quiz bittikten sonra liderlik pratiği (supervisor gözleminde)

### Mentörlük (Bar Buddy / Stajyer)
1. Sana mentörlük atandıysa → "Mentörlük Notları" widget
2. Haftada en az 1 değerlendirme yaz (5 dk)
3. Pratik becerileri puanla (1-5)
4. Geliştirilmesi gereken alanları yaz
5. Supervisor görür, terfi kararında kullanır

---

## 6. Acil Durumlar

| Durum | Aksiyon |
|---|---|
| Personel mesaiye gelmedi | Görev paylaştır, supervisor'a haber |
| Müşteri kavga ediyor | **Supervisor + mudur**'u hemen çağır, sen müdahil olma |
| Kalite checklist tamamlanamadı | Supervisor'a not + sebep |
| İzin talebi 3 günden uzun | **Onaylama**, supervisor'a yönlendir |
| Disiplin sorunu (personel) | Sen değerlendirme **yapma**, supervisor + mudur'a yönlendir |
| Mali konu (kasada açık) | **Asla kendi başına karar verme**, mudur'a haber |
| **Sistem yok** | Kâğıda yaz, sonra retroaktif gir |

## 9. Pilot Süresince Özel Notlar

- Pilot 4 lokasyonda **şu an 0 aktif supervisor buddy** var. Eğer Lara'da 3 baristadan biri terfi ederse bu cheat-sheet ilk doküman olmalı.
- İlk hafta supervisor ile **günde 1 değerlendirme** (sözlü, 10 dk) — vardiyada öğrendiklerin.
- Mentörlük atamaları pilot süresince supervisor manuel yapacak.

---

## 10. İlerleme Takibi

Telefondan **her hafta kontrol et**:
- "Benim Günüm" → "Supervisor Buddy → Supervisor" bar
- 7 modülden kaçı yeşil?
- Quiz ortalaman 85+ mı?
- Devam yüzden 90+ mı?
- Mentörlük puan ortalaman 4+ /5 mı?

5 alan da yeşilse supervisor'a "Supervisor sınavına hazırım" mesajı at.
