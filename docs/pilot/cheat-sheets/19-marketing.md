# DOSPRESSO Cheat Sheet — Marketing

**Hedef Kullanıcı**: Marketing ekibi — kampanya, içerik, sosyal medya, müşteri iletişim
**Cihaz**: Bilgisayar (asıl) + telefon (sosyal medya monitör)
**Erişim**: Marketing modülü, kampanya, içerik takvim, sosyal medya analitik

## 1. Marketing Nedir?

Sen DOSPRESSO marka algısını yönetirsin. CGO'ya bağlısın. Kampanya planla, içerik üret, sosyal medya yönet, müşteri iletişimini şekillendir.

## 4. Günlük İş Akışı

### Sabah
1. Sosyal medya etki dünkü → trend
2. Müşteri DM/yorum kontrolü
3. Bugün yayın planı

### Gün İçi
1. İçerik üret (görsel, metin, video)
2. Sosyal medya monitör (saatte bir)
3. Negatif yorum hızlı cevap (1 saat hedef)
4. Aktif kampanya performans

### Akşam
1. Bugünkü etki özet → CGO
2. Yarın yayın planı kontrol

### Haftalık
1. Pazartesi: hafta planı + kampanya onayları
2. Cuma: hafta sonu özel içerik

---

## 5. Sık Kullanılan İşler

### Yeni Kampanya Taslağı
1. Sidebar → "Kampanya" → "+ Yeni Taslak"
2. Hedef + kanal + bütçe + süre + KPI
3. İçerik (görsel + metin)
4. CGO onayına gönder
5. Onay sonrası → "Yayınla"

### Sosyal Medya Postu
1. Sidebar → "İçerik Takvimi" → "+ Post"
2. Kanal seç (Instagram / Facebook / TikTok)
3. Görsel + metin + hashtag
4. Yayın saati seç (peak zaman)
5. Onayla → otomatik yayınlama

### Müşteri DM/Yorum Cevap
1. "Müşteri İletişim" widget → bekleyen liste
2. Cevap yaz (marka tonu, kısa, samimi)
3. Negatif yorum → sorun sahibi şubeye yönlendir, müşteriye telafi
4. Çözüldü → "Kapalı" işaretle

### Etki Raporu
1. Sidebar → "Raporlar" → "Sosyal Etki"
2. Kanal × dönem
3. CGO'ya haftalık PDF

---

## 6. Acil Durumlar

| Durum | Aksiyon |
|---|---|
| Viral negatif yorum (binlerce paylaşım) | Acil: CGO + CEO brifing → resmi cevap |
| Yanlış post yayınlandı | Hemen sil + özür notu + neden açıkla |
| Marka kriz (sosyal medya kavga) | Sessiz mod (1 saat), CGO + hukuki danışman |
| Kampanya ters tepti | Acil durdur, performans düşüşü incelt |
| **Sosyal medya hesap hack** | Şifre değiştir + platform desteği + CGO |
