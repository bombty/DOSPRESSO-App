# DOSPRESSO Cheat Sheet — Stajyer

**Hedef Kullanıcı**: Yeni başlayan / işe alım sürecindeki kişi (kariyer merdiveninin 1. seviyesi)
**Cihaz**: Şube tableti (kiosk) + kendi telefonun (Benim Günüm — eğitim için)
**Erişim**: Sınırlı — gözlem + temel görevler + 7 zorunlu eğitim modülü

---

## 1. Stajyer Nedir?

Sen kariyer merdiveninin **1. seviyesindesin** (**Stajyer** → Bar Buddy → Barista → Supervisor Buddy → Supervisor).
Görevin: Sistemi öğren, ekibi izle, basit destek görevleri yap, 7 zorunlu modülü bitir, **30 günden sonra Bar Buddy'ye terfi et.**

**Önemli:** Bu seviyede henüz kasaya ve espresso makinesine doğrudan erişimin yok. Sadece supervisor/barista yanında pratik yapabilirsin.

### Yöntem A — Şube Tableti (Kiosk)
1. Tablet → "Şube Kiosk"
2. Adını listeden seç
3. PIN gir (4 haneli, sana verildi)
4. "Giriş" → Vardiya ekranı

### Yöntem B — Kendi Telefonun
1. Tarayıcıda sistem aç (link grup mesajında)
2. Kullanıcı adı: [size verilen, örn `stajyer_isiklar_1`]
3. Parola: SMS/WhatsApp ile geldi
4. "Giriş Yap" → "Benim Günüm"

---

## 3. Ana Ekran

### Kioskta (Şubede)
| Bölüm | İçerik |
|---|---|
| **Vardiya Durumu** | Giriş saati, kalan süre |
| **Görev Havuzu** | "Üstlen" butonlu kartlar — **sadece "genel" kategorisindeki basit işleri al** |
| **Bugünkü Skorum** | Tamamlanan + gecikmiş görevler |
| **Duyurular** | Vardiya başında oku |

### Telefonda (Benim Günüm)
| Bölüm | İçerik |
|---|---|
| **Eğitim İlerleme** | 7 zorunlu modül (hijyen, kahve temelleri, müşteri etiketi, vb.) |
| **Quiz'ler** | Modül sonu quiz'leri |
| **Stajyer → Bar Buddy İlerleme** | %0-100 bar, eşik 60 (her modülün quiz'i) |

---

## 4. Günlük İş Akışı (3 Adım)

### Vardiya Başı (08:00 / 16:00)
1. Kiosk girişi → adın + PIN
2. Mentor barista veya supervisor'a bugünkü plan sor
3. Açılış checklist'inde **gözlemci olarak** dur (önce izle, sonra yap)

### Vardiya Sırasında
1. Mentor'un yanında dur, müşteri akışını izle
2. Görev Havuzu'ndan "**genel**" kategorideki basit işler üstlen (örnek: vitrin temizliği, çöp, masa toplama)
3. Espresso makinesini **dokunma**, gözlem yap
4. Sorun gördüğünde mentor'a sor, **kendi başına karar verme**

### Vardiya Sonu
1. Kapanış işlerinde mentor'a yardım et
2. "Vardiya Bitir"
3. **Telefondan Benim Günüm'e gir** → günde 1 modül 20-30 dk çalış

---

## 5. Sık Kullanılan İşler

### Görev Üstlenme (Sadece Basit Olanlar)
1. Kiosk → "Şube Görev Havuzu"
2. Sadece kategori "**genel**" veya "**temizlik**" olan görevleri seç
3. "🙋 Üstlen" → "Tamamla" (foto isteniyorsa çek)

### Eğitim Modülü Çalışma (Telefondan)
1. Telefondan giriş → "Benim Günüm" → "Eğitimlerim"
2. Sıradaki modülü aç (sırayla git, atla yapma)
3. Video izle / metin oku → quiz çöz
4. **Geçme notu: 60** (Stajyer seviyesi)
5. Bittiğinde modül yeşil olur

### Bar Buddy'ye Terfi
1. **7 zorunlu modülün hepsi yeşil**
2. **30 gün** stajyer olarak çalış
3. **%85 devam yüzdesi** (vaktinde gelme)
4. Supervisor "Bar Buddy'ye hazırsın" değerlendirmesi → otomatik terfi 🎉

---

## 6. Acil Durumlar

| Durum | Aksiyon |
|---|---|
| Müşteri sana soru sordu, bilmiyorum | Kibarca "Hemen yardım çağırıyorum" → mentor barista çağır |
| Bardak kırıldı / döküldü | Hemen temizle, mentor'a haber, müşteriyi uyar |
| Eğitim modülü çok zor | Mentor'a sor, eğer çözmedi → supervisor'a mesaj |
| Kioskta yanlış görev üstlendim | Mentor'dan yardım al, beraber tamamlayın veya devret |
| **Mesaiye geç kalacağım** | Önce mudur'a sonra mentor'a haber (WhatsApp), sebep yaz |
| **Sistem yok** | Kâğıt kalem ile çalış, supervisor sonra retroaktif girer |
