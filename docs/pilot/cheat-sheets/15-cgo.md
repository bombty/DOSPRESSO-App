# DOSPRESSO Cheat Sheet — CGO (Chief Growth Officer)

**Hedef Kullanıcı**: Chief Growth Officer — büyüme stratejisi, marketing + satış + müşteri operasyonları sorumlusu
**Cihaz**: Bilgisayar (asıl) + telefon
**Erişim**: Marketing, CRM, satış raporları, kampanya yönetimi, müşteri analiz

---

## 1. CGO Nedir?

Sen DOSPRESSO'nun **büyüme motoru**sun. Marketing + Satış + Müşteri başarısını yönetirsin. Stratejik düzeyde CEO'ya, operasyonel düzeyde marketing + satınalma + CRM ekiplerine bağlısın.

Pilot süresince: **müşteri sentimenti + ciro trendi** senin sorumluluğunda. Pilot ilk hafta hedef: müşteri şikayet sayısı baseline'a göre +%10'u geçmesin.

## 4. Günlük İş Akışı

### Sabah
1. Dünkü ciro tüm şubeler — sapma kontrol
2. CRM şikayet listesi → kritik olanlar marketing/operasyon'a
3. Aktif kampanya performans

### Gün İçi
1. Marketing ekibinden kampanya brifingi (haftada 2)
2. Müşteri sentiment kontrol (saatlik değil, 2-3 saatte bir)
3. Pilot süresince: müdürler ile günde 1 sentiment brifing

### Akşam
1. Gün sonu özet — ciro vs hedef vs geçen hafta
2. Yarın için marketing önceliği

### Haftalık
1. Pazartesi: geçen hafta kampanya analiz + yeni kampanya planı
2. Cuma: hafta sonu öncesi promosyon kararı

---

## 5. Sık Kullanılan İşler

### Yeni Kampanya Başlatma
1. Sidebar → "Marketing" → "+ Yeni Kampanya"
2. Hedef (ciro artırma / yeni müşteri / sadakat)
3. Kanal (sosyal medya / SMS / kasa)
4. Bütçe + süre + KPI
5. CEO onayı (>10K TL bütçe için)
6. Yayınla → tüm şubelere otomatik brifing

### Müşteri Şikayet Eskalasyonu
1. CRM → "Açık Şikayetler" → kritik filtre
2. Şikayetin olduğu şube müdürüne yönlendir
3. 48 saat içinde çözüm bekle
4. Çözülmedi → CEO'ya brifing

### Şube Performans Kıyas
1. Sidebar → "Raporlar" → "Şube Kıyas"
2. Metrik: ciro / müşteri puan / personel verim
3. En kötü 3 şubeye coach atama
4. En iyi 3 şubeye ödül/öne çıkarma kararı

### NPS Skor Takibi
1. Dashboard "CRM Sentiment" widget
2. Aylık NPS trendi
3. <30 → müdahale gerekli (kalite, hizmet, ürün)
4. 50+ → marketing materyali olarak kullan

## 6. Acil Durumlar

| Durum | Aksiyon |
|---|---|
| Şubede viral şikayet (sosyal medya) | Marketing ekibi + müdür + CEO 1 saat içinde |
| Kampanya tepki çekti (negatif) | Acil durdur, marketing ekibi + CEO brifing |
| Müşteri sayısı %30+ düştü (haftalık) | Müdür + operasyon brifing, sebep analiz |
| Rakip ani fiyat indirimi | Pazarlama acil toplantı, fiyat stratejisi gözden geçir |
| **Sistem çökmesi** | IT eskalasyon, müşteri iletişimini WhatsApp grubuna manuel yap |
