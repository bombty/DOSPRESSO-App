# DOSPRESSO Cheat Sheet — Trainer

**Hedef Kullanıcı**: Eğitim sorumlusu — modül içerik, sınav, sertifika, beceri geliştirme
**Cihaz**: Bilgisayar (asıl) + tablet (saha eğitim)
**Erişim**: Akademi modülü, modül editörü, quiz, öğrenme yolları, sertifika

---

## 1. Trainer Nedir?

Sen DOSPRESSO ekibinin **gelişim mimarısın**. Akademi modüllerini yazar, kuiz hazırlar, sertifika programlarını tasarlar, saha eğitimi verirsin.

## 4. Günlük İş Akışı

### Sabah
1. Tamamlama oranı düşük modülleri kontrol
2. Quiz başarısı düşük olanlar
3. Eğitim talebi listesi

### Gün İçi
1. Yeni modül içerik üret (hafta 1-2 modül)
2. Quiz hazırla
3. Saha eğitimi (haftada 2-3 gün)

### Haftalık
1. Pazartesi: hafta plan
2. Coach ile koordineli (saha kim eğitim alacak)
3. Modül performans raporu CGO/CEO

---

## 5. Sık Kullanılan İşler

### Yeni Modül Oluşturma
1. Sidebar → "Modüller" → "+ Yeni"
2. Başlık + kategori + hedef rol
3. İçerik blokları (metin, video, görsel, quiz)
4. Geçme notu eşik (60 / 75 / 85 — role göre)
5. Taslak → coach onayı → yayın

### Quiz Hazırlama
1. Modül aç → "Quiz" sekmesi
2. Soru ekle (çoktan seçmeli, doğru/yanlış)
3. Doğru cevap işaretle
4. Soru bankası (10+ soru havuzdan random 5)
5. Süre limit + tekrar deneme hakkı

### Saha Eğitim (Şube/Fabrika)
1. Tablet → "Saha Eğitim" → "+ Yeni"
2. Şube seç + tarih + katılımcı listesi
3. Konu (örn espresso teknik, müşteri etiketi)
4. Eğitim sonrası: katılım onayı + foto + öneri
5. Sistem otomatik sertifika kaydı

### Sertifika Yenileme
1. Sidebar → "Sertifikalar" → yaklaşan yenileme
2. İlgili kişiye otomatik bildirim (30 gün öncesinden)
3. Yenileme modülü ata
4. Geçirdi → yenilendi
5. Geçemedi → tekrar dene veya ek eğitim

---

## 6. Acil Durumlar

| Durum | Aksiyon |
|---|---|
| Quiz cevapları sızdırıldı (sosyal medya) | Modül durdur + soru bankası yenile + araştırma |
| Modül içerik yasal yanlış (yeni düzenleme) | Acil revize + yayın + ilgililere yeniden eğitim |
| Saha eğitimde personel sakatlandı | İlk yardım + İK + müdür |
| Sertifika kayıt sistemi bozuldu | IT eskalasyon + kâğıt yedek |
| **Sistem yok** | Saha eğitim devam, sonra retroaktif kayıt |
