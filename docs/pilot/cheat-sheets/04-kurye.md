## 1. Login (Telefon)

1. Telefon tarayıcısında sistem adresini aç
2. **Tavsiye**: ana ekrana ekle ("Ana ekrana ekle" Chrome menüsü)
3. Kullanıcı adı: [size verilen username]
4. Parola: SMS ile geldi
5. "Giriş Yap" → mobil dashboard

---

## 2. Ana Ekran (Mobil)

Üstte 3 büyük buton:
- **🚲 Vardiya Başlat** (check-in)
- **📋 Aktif Siparişler**
- **✅ Vardiya Bitir** (check-out)

Aşağıda:
- Bugün tamamlanan teslimatlar (sayı)
- Sıradaki sipariş (varsa)
- Bildirim çanı

---

## 3. Günlük İş Akışı (3 Adım)

### Vardiya Başı
1. Lokasyona varınca: "🚲 Vardiya Başlat"
2. GPS doğrulama → otomatik check-in
3. Ekibe katıldığını bildir (otomatik)

### Sipariş Boyunca
1. "📋 Aktif Siparişler" → liste
2. Sipariş kartına bas → adres + müşteri bilgisi
3. "Yola çıktım" butonu → durum güncellenir
4. Teslim sonrası: "✅ Teslim Edildi" → fotoğraf çek (opsiyonel)
5. Tekrar liste → sıradaki sipariş

### Vardiya Sonu
1. "✅ Vardiya Bitir"
2. GPS doğrulama → check-out
3. Günün özeti otomatik kaydedilir

---

## 4. Sık Kullanılan İşler

### Sipariş Sorunu (Müşteri Yok / Adres Yanlış)
1. Sipariş kartı → "Sorun Bildir"
2. Sorun türü seç (müşteri yok, adres yanlış, vb.)
3. Açıklama yaz
4. Mudur'a otomatik bildirim gider

### Acil Durum (Kaza, Hastalık)
1. Bildirim çanı → "Acil Durum"
2. Lokasyon + sorun → mudur ve ekibe SMS

---

## 5. Acil Durumlar

| Durum | Aksiyon |
|---|---|
| Sipariş yetişmedi | Sipariş → "Sorun Bildir" → "Geç kaldım" |
