# DOSPRESSO Cheat Sheet — Supervisor

**Hedef Kullanıcı**: Vardiya sorumlusu (mudur'un sağ kolu)  
**Erişim**: Kendi vardiyası, görev yönetimi, kalite kontrol

---

## 1. Login Adımları

1. Tarayıcıda sistem aç
2. Kullanıcı adı: [size verilen, örn `supervisor_lara_1`]
3. Parola: SMS/WhatsApp ile geldi
4. "Giriş Yap" → Mission Control (Supervisor)

---

## 2. Ana Ekran

| Widget | İçerik |
|---|---|
| **Vardiya Özet** | Mesai başlangıcı, ekipte kim var, açık görevler |
| **Görev Akışı** | Sürmekte / Beklemede / Tamamlandı kartları |
| **Kalite Kontrol** | Bugün yapılması gereken denetim listesi (checklist) |
| **Hızlı Aksiyonlar** | "Görev Ata", "Vardiya Notu", "Müşteri Geri Bildirimi" |

---

## 3. Günlük İş Akışı (3 Adım)

### Vardiya Başı (08:00 / 16:00)
1. Vardiya açılış checklist: "Kalite Kontrol" → bugünkü liste
2. Ekibe görev ata: "+ Yeni Görev"
3. Önceki vardiyanın notlarını oku (sidebar → "Notlar")

### Vardiya Ortası
1. Görev kontrol: tamamlanmamış görevlere bakım
2. Stok hızlı sayım (eğer atandıysa)
3. Müşteri şikayet/öneri varsa CRM'e gir

### Vardiya Sonu
1. Açık görevleri sonraki vardiyaya devret
2. Gün sonu kalite checklist'i kapat
3. Vardiya notu yaz (sidebar → "Notlar" → "+ Yeni")

---

## 4. Sık Kullanılan İşler

### Yeni Görev Atama
1. Mission Control → "+ Yeni Görev"
2. Atanan kişi seç (barista, stajyer, vb.)
3. Açıklama yaz (1-2 cümle)
4. Son tarih: bugün/yarın
5. "Oluştur" → bildirim otomatik gider

### Çalışan İzin Talebi
1. Bildirim geldi mi? Sağ üst çan → "İzin Talebi"
2. Açıkla → onay/red butonu
3. Onay → mudur'a otomatik bildirim
4. Red → çalışana geri bildirim ver

### Müşteri Şikayet
1. Sidebar → CRM → "Yeni Şikayet"
2. Müşteri bilgisi (varsa) + sorun
3. "İletişime geçildi" işaretle
4. Çözüldüğünde "Çözüldü" + not
