# DOSPRESSO Cheat Sheet — Yatırımcı (HQ)

**Hedef Kullanıcı**: HQ tarafındaki yatırımcı — sistem genelinde mali + stratejik raporlara erişim, karar müdahale değil sadece izleme
**Cihaz**: Bilgisayar + telefon (mobil dashboard)
**Erişim**: Read-only — mali, KPI, stratejik raporlar (operasyonel müdahale yok)

---

## 1. Yatırımcı (HQ) Nedir?

Sen DOSPRESSO'da **finansal/stratejik paydaş**sın. Sisteme **read-only erişim**in var: rapor görür, KPI izler, brifinge katılır. Operasyonel kararlara müdahale **etmezsin** (CEO + CGO yetki).

---

## 2. Login Adımları

1. Tarayıcı → sistem aç
2. Kullanıcı adı: [size verilen, örn `yatirimci_hq`]
3. Parola: SMS/WhatsApp
4. "Giriş Yap" → Yatırımcı Dashboard

---

## 3. Ana Ekran (Yatırımcı Dashboard)

| Widget | İçerik |
|--------|--------|
| **Mali Özet** | Bugün, hafta, ay, yıl ciro + kar/zarar |
| **22 Şube + Fabrika Sağlık** | Genel performans renk kodu |
| **Pilot Durum** | Pilot kpi eşikleri, anlık sayı |
| **Stratejik Hedefler** | Çeyrek/yıl hedefleri vs gerçekleşen |
| **Risk Göstergeleri** | Kırmızı alarmlar (mali, operasyonel, yasal) |
| **Hisse/Pay Dağılım** | Yatırımcı bazlı pay (eğer çoklu) |

---

## 4. Günlük İş Akışı

### Sabah
1. Mali özet tara
2. Pilot durum (pilot süresince)
3. Risk göstergeleri (kırmızı yoksa devam)

### Haftalık
1. Pazartesi: hafta brifing (CEO + CGO ile zoom)
2. Hafta raporu PDF (otomatik mail)

### Aylık
1. Ay sonu mali rapor
2. Stratejik hedef gözden geçirme
3. Yönetim kurulu (varsa) brifingi

---

## 5. Sık Kullanılan İşler

### Mali Rapor İndirme
1. Sidebar → "Raporlar" → "Mali Özet"
2. Tarih aralığı seç
3. PDF/Excel indir

### Stratejik Hedef İzleme
1. Sidebar → "Stratejik Plan" → "Hedefler"
2. Çeyrek/yıl hedefleri vs gerçekleşen yüzde
3. Sapma sebepleri (CEO notları)

### Pilot Durum Takip (Pilot Süresince)
1. Dashboard "Pilot Durum" widget
2. 4 sayısal eşik anlık
3. Day-1 raporu (Salı akşamı)
4. Hafta raporu (5 May Pazartesi)

### Risk Göstergeleri Detay
1. "Risk Göstergeleri" widget → kırmızı alarm tıkla
2. Detay (sebep + ilgili müdahale)
3. CEO'ya soru ileti (chat veya mail)

---

## 6. Acil Durumlar

| Durum | Aksiyon |
|---|---|
| Mali sapma %20+ (haftalık) | CEO + CGO brifingi talep et |
| Pilot KPI eşik aşıldı | CEO ile bağlantı, brifing iste |
| Yasal sorun (denetim, dava) | CEO bilgisi al, kişisel müdahale yapma |
| Stratejik hedef ciddi sapma | Yönetim kurulu (varsa) toplantı |
| Sistem güvenlik olayı | IT + CEO bilgisi al |

**ÖNEMLİ:** Sen müdahale etmezsin, **bilgi alır + soru sorar + brifing istersin**.
