# DOSPRESSO Cheat Sheet — Gıda Mühendisi

**Hedef Kullanıcı**: Gıda Mühendisi (Sema) — reçete bilim, alerjen, beslenme, gıda güvenliği teknik
**Cihaz**: Bilgisayar (asıl) + tablet (saha)
**Erişim**: Reçete modülü, alerjen, beslenme veri, gıda güvenliği denetim
**Bağlı:** Reçete GM + Kalite Kontrol ile koordineli

---

## 1. Gıda Mühendisi Nedir?

Sen DOSPRESSO ürün bilimsel sorumlusun. Reçete + alerjen + beslenme bilgileri + gıda güvenliği teknik altyapısı sende. Reçete GM stratejik karar verir, sen **bilimsel/teknik temel** sağlarsın.

## 4. Günlük İş Akışı

### Sabah
1. Gün öncesi kalite raporu kontrolü
2. Yeni ürün talebi listesi
3. Yasal değişiklik haberleri tara

### Gün İçi
1. Reçete revizyon (Mr. recete_gm ile koordineli)
2. Alerjen güncellemesi
3. Beslenme etiket hesaplama
4. Saha denetim (haftada 1-2)

### Haftalık
1. Reçete GM ile haftalık değerlendirme
2. Kalite kontrol (umran) ile alerjen + risk paylaşım
3. Yeni ürün AR-GE (CGO talebi varsa)

---

## 5. Sık Kullanılan İşler

### Reçete Revizyon
1. Sidebar → "Reçete" → ürün seç
2. Mevcut formülasyon incele
3. Revizyon önerisi (yeni hammadde, oran değişikliği)
4. Beslenme + alerjen otomatik hesaplama
5. Reçete GM onayına gönder
6. Onay sonrası → menü/etiket otomatik güncellenir

### Alerjen Güncelleme
1. Sidebar → "Alerjen" → ürün seç
2. Alerjen ekle/çıkar (8 ana alerjen + ek)
3. Bilimsel gerekçe yaz (zorunlu)
4. Reçete GM + kalite kontrol onayı
5. Yayın → tüm menü/etiket güncellenir

### Yeni Ürün AR-GE
1. CGO/Marketing talebi → "Yeni Ürün" formu
2. Hedef profil (lezzet, fiyat, segment)
3. Formülasyon taslakları (3-5 alternatif)
4. Mutfak deneme + tat testi
5. Final reçete + maliyet + alerjen
6. Onay → pilot şube test → genel yayın

### Yasal Uyumluluk Takip
1. Sidebar → "Yasal" → güncel mevzuat
2. Türk Gıda Kodeksi değişiklikleri
3. Etkilenen ürünler listesi
4. Acil revizyon planla → reçete GM + kalite

## 6. Acil Durumlar

| Durum | Aksiyon |
|---|---|
| Müşteri alerjik reaksiyon raporu | Kalite kontrol (umran) + reçete GM + 24 saat araştırma |
| Yasal denetim ani gelişi | Reçete GM + kalite + tüm dokümanlar hazır mı acil kontrol |
| Hammadde formülasyon değiştirildi (tedarikçi) | Acil reçete revizyonu + alerjen güncel + onay |
| Yeni alerjen tespit (bilimsel makale) | Reçete GM brifing + alerjen tablosu güncel |
| Beslenme etiketi yanlış basıldı | Acil etiket revizyon + dağıtımdan toplama |
| **Sistem yok** | Kâğıt formla reçete revizyon, sonra retroaktif |
