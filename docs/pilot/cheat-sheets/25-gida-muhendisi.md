# DOSPRESSO Cheat Sheet — Gıda Mühendisi

**Hedef Kullanıcı**: Gıda güvenliği ve kalite uzmanı — Sema
**Cihaz**: Bilgisayar + Fabrika tablet
**Erişim**: HACCP, alerjen yönetimi, kalite kayıtları, fabrika ve şube denetim raporları

---

## 1. Gıda Mühendisi Nedir?

Sen DOSPRESSO'nun **gıda güvenliği + kalite uyum** sorumlususun. HACCP planı, alerjen kontrolü, hijyen denetimleri ve tüketici güvenliği senin alanın. Fabrika kalite kontrol'den sonra **son filtre** sensin.

---

1. Tarayıcı → sistem aç
2. Kullanıcı adı: `gida_muhendisi` (veya `sema`)
3. Parola: SMS/WhatsApp
4. "Giriş Yap" → Kalite Paneli

## 3. Ana Ekran Widget'ları

| Widget | İçerik |
|---|---|
| HACCP Uyum | Açık uygunsuzluk, CCP sapmaları |

---

## 4. Günlük Akış

**Sabah (09:00):**
1. HACCP Uyum widget → sapma varsa incele
2. Fabrika QC raporları → anomali tespit et
3. Şube denetim görevleri → değerlendirme

**Gün içi:**
4. Alerjen etiket güncelleme (reçete değişikliği)
5. CAPA (düzeltici aksiyon) takibi
6. Kalite şikayetleri → kök neden analizi

**Akşam:**
7. Günlük QC özeti hazırla
8. Kritik bulgu varsa CEO + Fabrika Müdürü'ne ilet

---

## 5. Sık Kullanılan İşler

**HACCP uygunsuzluk:**
1. Sidebar → "HACCP" → sapmalı CCP
2. Kök neden + düzeltici aksiyon yaz
3. CAPA aç → sorumlu ata
4. Takip: 24-72 saat (önem derecesine göre)

**Alerjen güncellemesi:**
1. Sidebar → "Alerjen" → ürün seç
2. Yeni alerjen ekle/çıkar
3. Onay → tüm menü/etiket otomatik güncellenir
4. Şubelere bildirim (trainer otomatik)

**Şube denetim sonucu:**
1. Sidebar → "Denetimler" → şube seç
2. Puan + bulgu yaz
3. Kritik bulgu: CAPA + 48 saat deadline
4. Takip raporu hazırla

**Gıda zehirlenmesi şüphesi (KRİTİK):**
1. Anında CEO + Fabrika Müdürü + Kalite Kontrol
2. Ürün partisi karantinaya al
3. Şubelere stop-sale duyurusu
4. Kök neden analizi + retroaktif lot takibi
