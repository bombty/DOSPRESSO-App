# DOSPRESSO Cheat Sheet — Coach (HQ)

**Hedef Kullanıcı**: HQ Coach — şube müdürlerini koçluk, performans gelişimi, vardiya planlama desteği, kariyer mentörlüğü
**Cihaz**: Bilgisayar (asıl) + telefon + saha ziyaret
**Erişim**: Tüm şubeler (read+write yetkili), vardiya planlama, performans, kariyer ilerleme

---

## 1. Coach Nedir?

Sen DOSPRESSO **müdürlerinin koçu**sun. Operasyonel sorumluluk değil, **gelişim sorumluluğu**. Müdür kendi şubesini yönetir, sen müdürün gelişimini izler, destekler, eksikleri tespit edersin.

Ek olarak HQ tarafında **vardiya planlamada tüm şubelere yetkili**sin (mudur sadece kendi şubesi).

## 4. Günlük İş Akışı

### Sabah
1. Müdür performans tarama → kırmızı/sarı şubeler
2. Saha ziyaret takvimi
3. Vardiya planlama eksikleri

### Gün İçi
1. Saha ziyaret (hafta içi 2-3 gün)
2. Müdür ile telefon brifingi (haftada her müdüre 1 kez)
3. Vardiya planlama desteği (mudur takılırsa)

### Haftalık
1. Pazartesi: hafta plan + en kötü 3 şube müdahale
2. Cuma: hafta sonu öncesi vardiya kontrol (tüm şubeler)
3. CGO/CEO'ya hafta raporu

---

## 5. Sık Kullanılan İşler

### Müdür Koçluk Görüşmesi
1. Sidebar → "Müdürler" → müdür seç
2. Performans + son ziyaret notları
3. Telefon/yüz yüze görüşme (30 dk)
4. Görüşme notu yaz (zorunlu)
5. Hedefler + eylem planı
6. Sonraki görüşme tarih

### Vardiya Planlama (Tüm Şubeler)
1. Sidebar → "Vardiya" → şube seç (veya "Tüm Şubeler")
2. Hafta tablo aç
3. Mudur taslağı varsa onayla / değiştir
4. Personel atama (drag-drop veya manual)
5. "Yayınla" → personele otomatik bildirim

### Kariyer Adayı Takip
1. Sidebar → "Kariyer" → aktif adaylar
2. Bar Buddy → Barista geçişleri
3. Supervisor Buddy → Supervisor geçişleri
4. Modül + pratik sınav durumu
5. Hazır olanlara terfi onayı (mudur ile koordineli)

### Saha Ziyaret
1. Sidebar → "Saha Ziyaret" → "+ Yeni"
2. Şube + tarih + öncelik konuları
3. Yerinde gözlem + müdür ile değerlendirme
4. Notlar (zorunlu) + foto
5. Eylem planı + sonraki ziyaret

## 6. Acil Durumlar

| Durum | Aksiyon |
|---|---|
| Müdür ekipte ciddi çatışma | Acil saha gidiş + İK koordineli |
| Şube performans haftalık %30+ düştü | Acil saha + müdür koçluk + müdahale planı |
| Müdür istifa talebi | İK + CEO acil brifing |
| Vardiya planı tamamlanamadı (deadline) | Sen planlamayı al, müdür ile sonra görüş |
| Kariyer adayı pratik sınav başarısız | Trainer + müdür koordineli ek eğitim |
| **Sistem yok** | Telefon ile koordineli, sonra retroaktif |
