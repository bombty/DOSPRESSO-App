## ⚠ KRİTİK
- Şifre bcrypt ile hashle (mevcut seed pattern'ini kullan)
- users.id VARCHAR UUID formatında olmalı
- factory_staff_pins'e de PIN ekle (kiosk girişi için)
- Do NOT touch localAuth.ts, vite.ts, vite.config.ts

---

## ADIM 1: Üretim Şefi kullanıcı oluştur

```sql
-- Önce mevcut uretim_sefi rolü var mı kontrol et
SELECT name, display_name FROM roles WHERE name = 'uretim_sefi';

-- Kullanıcı oluştur (bcrypt hash gerekli — kod ile yap)
```

**Kod ile kullanıcı oluştur** (bcrypt hash için):

```typescript
// server tarafında çalıştır veya seed script'ine ekle
import bcrypt from 'bcryptjs';
import { v4 as uuidv4 } from 'uuid';

const userId = uuidv4();
const hashedPassword = await bcrypt.hash('0000', 10);

await db.insert(users).values({
  id: userId,
  username: 'umit',
  password: hashedPassword,
  firstName: 'Ümit',
  lastName: 'Usta',
  role: 'uretim_sefi',
  branchId: 24,          // Fabrika branch ID
  isActive: true,
  email: null,
  phone: null,
});

// Kiosk PIN ekle
await db.insert(factoryStaffPins).values({
  userId: userId,
  pin: '5555',           // 4 haneli PIN (kiosk girişi için)
});

console.log('Üretim Şefi oluşturuldu:', userId);
```

**Veya mevcut seed pattern'ini kullanarak ekle.** Önemli olan:
1. role = 'uretim_sefi'
2. branchId = 24 (Fabrika)
3. Şifre bcrypt hash'li
4. factory_staff_pins'te PIN kaydı

## ADIM 4: Doğrulama

```bash
echo "=== Üretim Şefi doğrulama ==="

echo "--- Kullanıcı mevcut mu? ---"
echo "SELECT id, username, first_name, last_name, role, branch_id FROM users WHERE username='umit';" | psql

echo "--- Login testi ---"
RESULT=$(curl -s -X POST http://localhost:5000/api/login -H "Content-Type: application/json" -d '{"username":"umit","password":"0000"}' -c /tmp/umit.txt)
echo "Login: $RESULT" | head -c 200

echo ""
echo "--- Menü ---"
curl -s http://localhost:5000/api/me/menu -b /tmp/umit.txt | node -e "
  const d=JSON.parse(require('fs').readFileSync('/dev/stdin','utf8'));
  d.forEach(s=>{if(s.items)s.items.forEach(i=>console.log('  '+i.label+' → '+(i.path||'')))});
" 2>/dev/null

echo ""
echo "--- Kiosk PIN ---"
echo "SELECT user_id, pin FROM factory_staff_pins WHERE user_id = (SELECT id FROM users WHERE username='umit');" | psql

echo ""
echo "--- Ürün sorumlulukları ---"
echo "SELECT pr.product_category, u.first_name, u.last_name, u.role 
FROM production_responsibilities pr 
JOIN users u ON pr.user_id = u.id 
WHERE pr.is_active = true
ORDER BY u.role, pr.product_category;" | psql

echo ""
echo "--- Üretim planlama erişimi ---"
curl -s -o /dev/null -w "Plans API: %{http_code}\n" -b /tmp/umit.txt "http://localhost:5000/api/production-planning/plans"

echo ""
echo "--- Fabrika Müdürü sorumlulukları ---"
echo "SELECT pr.product_category FROM production_responsibilities pr 
JOIN users u ON pr.user_id = u.id 
WHERE u.role = 'fabrika_mudur' AND pr.is_active = true
ORDER BY pr.product_category;" | psql

echo ""
echo "Server: $(curl -s -o /dev/null -w '%{http_code}' http://localhost:5000/)"
```
