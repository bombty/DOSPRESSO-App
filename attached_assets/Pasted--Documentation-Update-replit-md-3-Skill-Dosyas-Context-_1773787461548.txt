```markdown
## 10. Kiosk Auth Failures

### Symptom: 401 on kiosk endpoints
- Kiosk endpoints use isKioskAuthenticated (not isAuthenticated)
- Check x-kiosk-token header is being sent
- Kiosk session stored in-memory — lost on server restart
- Fix: worker must re-login with PIN after server restart

### Symptom: PIN always fails
- Check factory_staff_pins / branch_staff_pins table
- PIN must be bcrypt hashed — plain text rejected
- Check pinLockedUntil — may be locked for 15 min
- Fix: POST /api/factory/pins/:userId/unlock (admin only)

### Symptom: Device password rejected
- Check factory_kiosk_config or branch_kiosk_settings
- Password must be bcrypt hash — plain text rejected
- migrateKioskPasswords() runs on startup and hashes plain text
- Default password: "0000"

### Symptom: KioskRoleGuard redirecting everyone
- KioskRoleGuard only affects: fabrika_operator, fabrika_personel, 
  fabrika_sorumlu, sube_kiosk roles
- /login path must be excluded from redirect
- If ALL users are trapped: remove KioskRoleGuard wrapper, it's too aggressive

## 11. SLA Business Hours Issues

### Symptom: SLA deadline is wrong timezone
- Uses Europe/Istanbul via addBusinessHours() in business-hours.ts
- Check sla_business_hours table has exactly 1 row
- Default: Mon-Fri 08:00-18:00

### Symptom: SLA breach notification on weekend
- checkSlaBreaches() checks isBusinessHours() before sending
- Off-hours breaches deferred to next business day

### Symptom: SLA rules not loading
- Check sla_rules table has 24 rows (6 depts × 4 priorities)
- seedSlaRules() runs on startup — check server logs

## 12. Delegation System Issues

### Symptom: User has role but can't access module
- Check module_delegations table for active delegations
- GET /api/delegations/active returns delegations for current user's role
- Frontend checks this and adds ModuleCards to dashboard

### Symptom: Delegation not showing in CRM nav
- CRM nav reads delegatedDepts from active delegations
- Check iletisim-merkezi/index.tsx for delegation query
```

```markdown
## 11. Kiosk Auth Check
- New kiosk endpoints use isKioskAuthenticated (not isAuthenticated)?
  grep -n "isKioskAuthenticated" server/routes/factory.ts | head -5
- No kiosk endpoint uses plain isAuthenticated?

## 12. bcrypt Password Security
- New user/kiosk accounts have bcrypt hashed passwords?
  Never store plain text passwords or PINs
- New seed scripts use bcrypt.hash() before inserting?
  grep -n "bcrypt\|hash" server/routes/seed.ts | head -10

## 13. SLA Business Hours Consistency
- sla_rules table has 24 rows after any migration?
  SELECT COUNT(*) FROM sla_rules; — Expected: 24
- sla_business_hours has 1 row?
  SELECT COUNT(*) FROM sla_business_hours; — Expected: 1

## 14. CRM Endpoint Auth
- All crm-iletisim.ts endpoints have auth middleware?
  grep -n "router\." server/routes/crm-iletisim.ts | wc -l
  grep -n "isAuthenticated\|requireAuth" server/routes/crm-iletisim.ts | wc -l
  Both counts should be similar

## 15. Kiosk Role Guard Safety
- KioskRoleGuard allows /login path?
  grep -n "login" in KioskRoleGuard function
- sube_kiosk + fabrika_operator users redirect to correct kiosk?
```
