### EK TASK — T011: Gıda Mühendisi Fabrika Entegrasyonu (~2-3 saat)

Gıda Mühendisi (sema) fabrika üretim süreçlerinde kritik rol oynuyor. Kalite, gıda güvenliği, reçete doğrulama ve HACCP uyum sorumluluğu var.

**A) Gıda Mühendisi Fabrika Yetkileri:**

Mevcut gida_muhendisi rolüne fabrika modül izinleri ekle:
```bash
grep -rn "gida_muhendisi\|gıda.*mühendis" shared/schema.ts server/menu-service.ts --include="*.ts" | head -15
```

PERMISSIONS matrisine ekle:
- factory_dashboard: view

- factory_kiosk: view

- factory_analytics: view

- production_lots: view, edit (lot statüsü değiştirme — recall, hold)

SIDEBAR_ALLOWED_ITEMS'a gida_muhendisi için fabrika menü öğelerini ekle.

**B) Kalite Kontrol Genişletme (Gıda Mühendisi Odaklı):**

Mevcut kalite kontrol sistemi basit approve/reject. Gıda mühendisi için genişlet:

1. **Kalite kontrol formu genişletme:**
```sql
ALTER TABLE factory_quality_checks ADD COLUMN IF NOT EXISTS 
  visual_inspection VARCHAR(20), -- 'pass', 'fail', 'conditional'
  taste_test VARCHAR(20),
  texture_check VARCHAR(20),
  weight_check VARCHAR(20),
  temperature_check VARCHAR(20),
  packaging_integrity VARCHAR(20),
  allergen_check BOOLEAN DEFAULT false,
  haccp_compliance BOOLEAN DEFAULT true,
  inspector_notes TEXT,
  corrective_action TEXT, -- Red durumunda düzeltici faaliyet
  hold_reason TEXT, -- Bekletme sebebi
  food_engineer_approval BOOLEAN DEFAULT false,
  food_engineer_id VARCHAR(255) REFERENCES users(id),
  food_engineer_approved_at TIMESTAMP;
```

2. **Kalite kontrol akışı (2 aşamalı):**
```
Operatör üretir → Kalite teknisyeni ilk kontrol (görsel, ağırlık, ambalaj)
  → PASS → Gıda mühendisi final onay (tat, doku, HACCP uyum)
    → APPROVED → Stoka ekle + LOT aktif
    → HOLD → Beklemede (ek test gerekli) + Gıda mühendisine bildirim
    → REJECTED → Fire + düzeltici faaliyet kaydı + bildirim
  → FAIL → Direkt red + gıda mühendisine bildirim
```

3. **Kritik ürünlerde gıda mühendisi onayı ZORUNLU:**
   - Cheesecake, cookie, kek, tuzlu ürünler → gıda mühendisi final onayı şart
   - Donut, konsantre, toz, alsat → teknisyen onayı yeterli
   - Kahve kavurma → gıda mühendisi kavurma kalite puanını onaylar

```sql
ALTER TABLE factory_products ADD COLUMN IF NOT EXISTS 
  requires_food_engineer_approval BOOLEAN DEFAULT false;
```
Cheesecake, cookie, kek, tuzlu, kahve ürünlerinde true olarak set et.

**C) Gıda Mühendisi Dashboard Widget'ları:**

Gıda mühendisi dashboard'una fabrika widget'ları ekle:

1. **"Bekleyen Kalite Onayları" widget:**
   - Gıda mühendisi onayı bekleyen lot sayısı
   - Acil olanlar (SKT yakın) kırmızı
   - Tıklayınca kalite kontrol sayfasına git

2. **"Günlük Üretim Özeti" widget:**
   - Bugün kaç lot üretildi
   - Kaçı onaylandı, kaçı beklemede, kaçı red
   - Fire oranı (bugün ve son 7 gün trendi)

3. **"SKT Uyarıları" widget:**
   - 3 gün içinde SKT dolacak lotlar
   - Süresi geçmiş ama hâlâ stokta olan lotlar (KRİTİK kırmızı)

4. **"HACCP Uyum" widget:**
   - Son 30 günde HACCP uyumsuzluk sayısı
   - Açık düzeltici faaliyetler

**D) Gıda Güvenliği Kayıtları:**

1. **HACCP kontrol noktası kayıtları:**
```sql
CREATE TABLE IF NOT EXISTS haccp_check_records (
  id SERIAL PRIMARY KEY,
  check_point VARCHAR(100) NOT NULL, -- 'sicaklik_kontrol', 'hijyen_kontrol', 'allerjen_kontrol', 'cross_kontaminasyon'
  station_id INTEGER REFERENCES factory_stations(id),
  checked_by VARCHAR(255) REFERENCES users(id), -- Genellikle gıda mühendisi
  check_date TIMESTAMP DEFAULT NOW(),
  result VARCHAR(20) NOT NULL, -- 'pass', 'fail', 'warning'
  temperature_value DECIMAL(5,2), -- Sıcaklık ölçümü (varsa)
  corrective_action TEXT, -- Fail durumunda ne yapıldı
  notes TEXT,
  lot_id INTEGER REFERENCES production_lots(id), -- Hangi lot ile ilgili
  created_at TIMESTAMP DEFAULT NOW()
);
```

2. **HACCP endpoint'leri:**
```
POST /api/factory/haccp — Yeni HACCP kaydı
GET /api/factory/haccp — Liste (filtreli: tarih, istasyon, sonuç)
GET /api/factory/haccp/summary — Özet (son 30 gün pass/fail/warning sayıları)
```

3. **Fabrika mega-module'e "Gıda Güvenliği" tab'ı ekle (veya mevcut Kalite Kontrol tab'ını genişlet):**
   - HACCP kontrol noktası kayıt formu
   - Sıcaklık kaydı (soğuk hava, şoklama, üretim alanı)
   - Hijyen kontrol listesi
   - Düzeltici faaliyet takibi
