## BÖLÜM A: DATABASE (⚠ DANIŞMA — SQL göster)

### A1: user_todos tablosu

```sql
CREATE TABLE IF NOT EXISTS user_todos (
  id SERIAL PRIMARY KEY,
  user_id VARCHAR(36) NOT NULL REFERENCES users(id),
  title VARCHAR(300) NOT NULL,
  description TEXT,
  due_date TIMESTAMP,
  due_time VARCHAR(5),           -- '14:30' formatında (opsiyonel saat)
  priority VARCHAR(20) DEFAULT 'medium',  -- low, medium, high, urgent
  status VARCHAR(20) DEFAULT 'pending',   -- pending, in_progress, done, cancelled
  source VARCHAR(30) DEFAULT 'manual',    -- manual, crm_ticket, equipment, system, dobody, branch_task
  source_id VARCHAR(50),                  -- ilgili ticket/arıza/görev ID'si
  source_url VARCHAR(200),               -- deep link (tıklanınca kaynağa git)
  tags TEXT[],                            -- ['tedarikci', 'ahmet', 'numune']
  is_recurring BOOLEAN DEFAULT false,
  recurrence_pattern VARCHAR(30),         -- daily, weekly, monthly, yearly
  completed_at TIMESTAMP,
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);

CREATE INDEX IF NOT EXISTS idx_user_todos_user ON user_todos(user_id, status, due_date);
CREATE INDEX IF NOT EXISTS idx_user_todos_source ON user_todos(source, source_id);
```

### A2: user_calendar_events tablosu

```sql
CREATE TABLE IF NOT EXISTS user_calendar_events (
  id SERIAL PRIMARY KEY,
  user_id VARCHAR(36) NOT NULL REFERENCES users(id),
  title VARCHAR(300) NOT NULL,
  description TEXT,
  start_time TIMESTAMP NOT NULL,
  end_time TIMESTAMP,
  all_day BOOLEAN DEFAULT false,
  event_type VARCHAR(30) DEFAULT 'meeting',  -- meeting, reminder, deadline, visit, call, training, shift
  location TEXT,
  color VARCHAR(20),                          -- UI renk kodu (opsiyonel)
  related_entity_type VARCHAR(30),           -- branch, supplier, user, ticket, equipment
  related_entity_id VARCHAR(50),
  related_entity_url VARCHAR(200),           -- deep link
  recurrence VARCHAR(30) DEFAULT 'none',     -- none, daily, weekly, monthly, yearly
  reminder_minutes INTEGER DEFAULT 30,        -- kaç dk önce hatırlat
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);

CREATE INDEX IF NOT EXISTS idx_user_calendar_user ON user_calendar_events(user_id, start_time);
```

### A3: user_notes tablosu

```sql
CREATE TABLE IF NOT EXISTS user_notes (
  id SERIAL PRIMARY KEY,
  user_id VARCHAR(36) NOT NULL REFERENCES users(id),
  title VARCHAR(300),
  content TEXT NOT NULL,
  related_entity_type VARCHAR(30),   -- branch, supplier, user, equipment, ticket, product
  related_entity_id VARCHAR(50),
  related_entity_name VARCHAR(200),  -- gösterim için (ör: "Ahmet Tedarikçi", "Işıklar Şubesi")
  tags TEXT[],
  is_pinned BOOLEAN DEFAULT false,
  color VARCHAR(20),                 -- not kartı rengi
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);

CREATE INDEX IF NOT EXISTS idx_user_notes_user ON user_notes(user_id, is_pinned DESC, updated_at DESC);
```

```

  ?status=pending|done|all

  ?priority=low|medium|high|urgent

  { title, description?, dueDate?, dueTime?, priority?, tags?[] }

DELETE /api/ajanda/todos/:id                — sil (soft delete veya hard)

POST   /api/ajanda/todos/:id/complete       — tamamla (kısayol)
```

## BÖLÜM D: MR. DOBODY SKILL'LERİ

### D1: Sabah Briefing skill — `daily-briefing`

Mevcut skill pattern'ini takip et:

```typescript
// server/services/skills/daily-briefing.ts
// Trigger: Her gün sabah 07:00
// Hedef: Kullanıcının bugünkü özeti

action: async (userId) => {
  const todayTodos = await getTodayTodos(userId);
  const todayEvents = await getTodayEvents(userId);
  const openTickets = await getUserOpenTickets(userId);
  const overdueTodos = todayTodos.filter(t => isOverdue(t));
  
  const messages = [];
  
  if (todayEvents.length > 0)
    messages.push(`${todayEvents.length} toplantı/etkinlik`);
  if (todayTodos.length > 0)
    messages.push(`${todayTodos.length} yapılacak`);
  if (overdueTodos.length > 0)
    messages.push(`${overdueTodos.length} gecikmiş görev`);
  if (openTickets.slaNearing > 0)
    messages.push(`${openTickets.slaNearing} SLA riski`);
  
  if (messages.length > 0) {
    await createAgentAction({
      userId,
      skillId: 'daily-briefing',
      type: 'briefing',
      title: 'Günaydın! Bugünün özeti',
      message: `Bugün: ${messages.join(', ')}`,
      deepLink: '/ajanda',
      priority: overdueTodos.length > 0 ? 'high' : 'medium',
      expiresAt: endOfDay()
    });
  }
}
```

### D2: Akıllı Hatırlatma skill — `smart-reminder`

```typescript
// Trigger: Her 2 saatte bir (veya event reminder_minutes öncesinde)
// 1. Yaklaşan etkinlikler (30dk/1saat önce)
// 2. Bugün bitmesi gereken todo'lar (öğleden sonra hatırlatma)
// 3. Gecikmiş todo'lar (günlük nudge)
// 4. Uzun süredir güncellenmemiş notlar ("Ahmet tedarikçi notu 7 gündür güncellenmedi")

action: async () => {
  // Tüm kullanıcıları tara (aktif olanlar)
  const activeUsers = await getActiveUsers();
  
  for (const user of activeUsers) {
    // Yaklaşan etkinlikler
    const upcomingEvents = await getEventsInRange(user.id, now, now + 30min);
    for (const event of upcomingEvents) {
      await createNotification({
        userId: user.id,
        type: 'calendar_reminder',
        title: `Yaklaşan: ${event.title}`,
        message: `${formatTime(event.startTime)} — ${event.location || ''}`,
        deepLink: '/ajanda',
        priority: 'medium'
      });
    }
    
    // Gecikmiş todo'lar (günde 1 kez, sabah)
    if (isMorning()) {
      const overdue = await getOverdueTodos(user.id);
      if (overdue.length > 0) {
        await createAgentAction({
          userId: user.id,
          skillId: 'smart-reminder',
          type: 'nudge',
          title: `${overdue.length} gecikmiş görev`,
          message: `En eski: "${overdue[0].title}" — ${daysOverdue(overdue[0])} gün gecikmiş`,
          deepLink: '/ajanda?view=todos&filter=overdue',
          priority: 'high'
        });
      }
    }
  }
}
```

### D3: Doğal Dil Todo skill — `natural-language-todo`

**Bu skill Mr. Dobody chat'inden tetiklenir (kullanıcı yazarak komut verir):**

```typescript
// Kullanıcı Mr. Dobody'ye yazıyor: "Yarın Ahmet'i ara"
// Agent bunu parse edip todo + takvim oluşturuyor

// Pattern matching (basit regex — GPT'ye göndermeye gerek yok):
const patterns = [
  // "yarın X yap" → todo (due: yarın)
  { regex: /yar[ıi]n\s+(.+)/i, action: 'todo', dueOffset: 1 },
  // "cuma X" → todo (due: önümüzdeki cuma)
  { regex: /(pazartesi|sal[ıi]|[çc]ar[şs]amba|per[şs]embe|cuma|cumartesi|pazar)\s+(.+)/i, action: 'todo' },
  // "saat 14:00 X" → event
  { regex: /saat\s+(\d{1,2}[:.]\d{2})\s+(.+)/i, action: 'event' },
  // "hatırlat: X" → reminder
  { regex: /hat[ıi]rlat[:\s]+(.+)/i, action: 'reminder' },
  // "not: X" → note
  { regex: /not[:\s]+(.+)/i, action: 'note' },
];

// Mr. Dobody chat handler'ında:
// 1. Kullanıcı mesajını pattern'lere karşı test et
// 2. Eşleşme varsa → ilgili API'yi çağır (todo/event/note oluştur)
// 3. Onay mesajı dön: "Yarın 'Ahmet'i ara' todo'su eklendi ✓"
// 4. Eşleşme yoksa → normal AI chat devam etsin
```

**⚠ Bu skill mevcut Mr. Dobody chat handler'ına hook olarak eklenmeli. Chat handler'ı bul, mesaj işlenmeden ÖNCE pattern check yap. Eşleşirse todo/event/note oluştur ve kısa onay dön. Eşleşmezse normal AI akışına devam et.**
