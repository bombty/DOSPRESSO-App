### 1B: Lot number generator utility

Create `server/utils/lot-number.ts`:
```typescript
// Format: LOT-YYYYMMDD-XXX (sequential per day)
// Example: LOT-20260326-001

export async function generateLotNumber(db: any, date: Date): Promise<string> {
  const dateStr = date.toISOString().split('T')[0].replace(/-/g, '');
  const prefix = `LOT-${dateStr}-`;
  
  const existing = await db.select({ lotNumber: productionLots.lotNumber })
    .from(productionLots)
    .where(like(productionLots.lotNumber, `${prefix}%`))
    .orderBy(desc(productionLots.lotNumber))
    .limit(1);
  
  let nextNum = 1;
  if (existing.length > 0) {
    const lastNum = parseInt(existing[0].lotNumber.split('-').pop() || '0');
    nextNum = lastNum + 1;
  }
  
  return `${prefix}${nextNum.toString().padStart(3, '0')}`;
}
```

### 2B: Create consolidated payroll service

Create `server/services/payroll-calculation-service.ts`:

This is THE ONE engine. Key functions:

```typescript
// Main function
export async function calculateMonthlyPayroll(db, input: PayrollInput): Promise<PayrollResult>

// Sub-functions
function calculateBaseSalary(grossMonthly: number, workedDays: number, expectedDays: number): number
function calculateOvertime(overtimeMinutes: number, hourlyGross: number, params: PayrollParams): number
function calculateSGK(grossTotal: number, params: PayrollParams): { employee: number, employer: number }
function calculateIncomeTax(taxableIncome: number, cumulativeBase: number, params: PayrollParams): number
function calculateStampTax(grossTotal: number, params: PayrollParams): number
function calculateAGI(maritalStatus: string, childCount: number, spouseWorking: boolean, params: PayrollParams): number

// Helper
async function getActivePayrollParams(db): Promise<PayrollParams>
async function getEmployeeCumulativeTaxBase(db, userId: string, year: number, upToMonth: number): Promise<number>
```

**Turkish labor law rules embedded:**
1. Monthly salary always 30-day basis (even Feb 28)
2. Daily gross = monthly / 30, hourly = daily / 7.5
3. Overtime: normal day 1.5x, OFF day 1.5x (all hours), holiday 2.0x
4. Holiday not worked: paid as normal (no extra)
5. SGK ceiling check: if gross > ceiling, SGK on ceiling only
6. Income tax: CUMULATIVE brackets (must track YTD)
7. Damga vergisi: on total gross
8. AGI: based on family status (read from payroll_parameters)
9. Sick leave: SGK pays, employer doesn't (different handling)
10. Unauthorized absence: daily deduction = gross / 30

**Read all rates from payroll_parameters table — NEVER hardcode:**
```typescript
// Decode integer-stored values
const params = {
```
