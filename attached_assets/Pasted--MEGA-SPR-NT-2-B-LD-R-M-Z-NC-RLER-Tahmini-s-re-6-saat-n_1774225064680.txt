## ADIM 1: FABRİKA — Üretim Kaydı Bildirimi (30 dk)

**Sorun:** factory.ts'deki log-production endpoint'i üretim kaydı girildiğinde fabrika müdürüne hiç bildirim göndermiyor. 128 endpoint, 0 üretim bildirimi.

**Bul:** factory.ts'de log-production POST endpoint'ini bul.

```bash
grep -n "log-production\|logProduction\|log_production" server/routes/factory.ts | head -10
```

**Ekle:** INSERT başarılı olduktan sonra, createNotification çağrısı ekle:

```typescript
// Üretim kaydı başarılı INSERT'ten sonra ekle:

// 1. Fabrika müdürüne bildirim
const fabrikaMudurler = await db.select({ id: users.id })
  .from(users)
  .where(and(eq(users.role, 'fabrika_mudur'), eq(users.isActive, true)));

for (const mudur of fabrikaMudurler) {
  await createNotification({
    userId: mudur.id,
    type: 'factory_production_log',
    title: 'Yeni üretim kaydı',
    message: `${userName} — ${productName}: ${quantity} ${unit} üretildi${wasteQuantity > 0 ? `, ${wasteQuantity} ${wasteUnit} fire` : ''}`,
    severity: 'info',
    deepLink: '/fabrika/uretim-planlama',
  });
}
```

**NOT:** Her üretim kaydında bildirim çok fazla olabilir. Alternatif: sadece fire oranı yüksekse (%5+) bildirim gönder. Karar: İKİSİNİ DE YAP — normal üretim = info, yüksek fire = warning.

```typescript
// Fire oranı kontrolü
const wasteRatio = wasteQuantity / (quantity + wasteQuantity);
if (wasteRatio > 0.05) {
  // %5'ten fazla fire → UYARI seviyesiyle gönder
  for (const mudur of fabrikaMudurler) {
    await createNotification({
      userId: mudur.id,
      type: 'factory_high_waste_alert',
      title: 'Yüksek fire uyarısı',
      message: `${userName} — ${productName}: %${(wasteRatio * 100).toFixed(1)} fire oranı (${wasteQuantity} ${wasteUnit})`,
      severity: 'warning',
      deepLink: '/fabrika/uretim-planlama',
    });
  }
  
  // Gıda mühendisine de gönder
  const gidaMuhendisleri = await db.select({ id: users.id })
    .from(users)
    .where(and(eq(users.role, 'gida_muhendisi'), eq(users.isActive, true)));
  
  for (const muh of gidaMuhendisleri) {
    await createNotification({
      userId: muh.id,
      type: 'factory_high_waste_alert',
      title: 'Yüksek fire uyarısı',
      message: `${userName} — ${productName}: %${(wasteRatio * 100).toFixed(1)} fire oranı`,
      severity: 'warning',
      deepLink: '/fabrika/uretim-planlama',
    });
  }
}
```

**Test:** Bir üretim kaydı gir → notifications tablosunda factory_production_log görünmeli.

**Sorun:** QC denetimi yapıldığında (approved/rejected) ne fabrika müdürü ne de ilgili operatör bilgilendirilmiyor.

**Bul:** factory.ts'de QC inspection/decision endpoint'ini bul.

```bash
grep -n "quality.*check\|qc.*inspect\|qc.*decision\|inspection\|quality_check" server/routes/factory.ts | grep -i "post\|patch\|put" | head -10
```

**Ekle:** QC kararı (approved/rejected/pending_engineer) kaydedildikten sonra:

```typescript
// QC sonucu bildirimi

// 1. Üreten operatöre bildir
if (producerId) {
  await createNotification({
    userId: producerId,
    type: 'qc_result',
    title: decision === 'approved' ? 'QC onaylandı' : decision === 'rejected' ? 'QC reddedildi' : 'Gıda mühendisi incelemesine yönlendirildi',
    message: `${productName} (LOT: ${lotNumber || 'N/A'}) — ${decisionReason || ''}`,
    severity: decision === 'rejected' ? 'warning' : 'info',
    deepLink: '/fabrika/kalite-kontrol',
  });
}

// 2. Fabrika müdürüne bildir
for (const mudur of fabrikaMudurler) {
  await createNotification({
    userId: mudur.id,
    type: 'qc_result',
    title: `QC: ${decision === 'approved' ? 'Onaylandı' : decision === 'rejected' ? 'Reddedildi' : 'Mühendise yönlendirildi'}`,
    message: `${productName} — Denetçi: ${inspectorName}`,
    severity: decision === 'rejected' ? 'warning' : 'info',
    deepLink: '/fabrika/kalite-kontrol',
  });
}

// 3. Gıda mühendisine yönlendirildiyse
if (decision === 'pending_engineer') {
  const gidaMuhendisleri = await db.select({ id: users.id })
    .from(users)
    .where(and(eq(users.role, 'gida_muhendisi'), eq(users.isActive, true)));
  
  for (const muh of gidaMuhendisleri) {
    await createNotification({
      userId: muh.id,
      type: 'qc_engineer_review',
      title: 'QC inceleme bekliyor',
      message: `${productName} (LOT: ${lotNumber || 'N/A'}) — gıda mühendisi onayı gerekiyor`,
      severity: 'warning',
      deepLink: '/fabrika/kalite-kontrol',
    });
  }
}
```
