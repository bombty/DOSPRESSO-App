MODE: BACKGROUND PROJECT TASK REQUIRED

Bu işlem main agent tarafından yapılamıyor çünkü git rm / commit / push destructive git operation sayılıyor.
Lütfen bunu izole background task agent olarak yap.

Amaç:
Git history rewrite yapmadan ve force push yapmadan:
1) local main ile origin/main senkronize edilsin,
2) hassas attached_assets dosyaları sadece Git tracking’den çıkarılsın,
3) .gitignore koruması doğrulansın,
4) normal push ile origin/main güncellensin.

KURALLAR:
- Force push YAPMA.
- git filter-repo YAPMA.
- git filter-branch YAPMA.
- history rewrite YAPMA.
- Dosyaları workspace’ten fiziksel olarak silme.
- DB, schema, migration, env, package.json, package-lock.json değiştirme.
- Kod dosyalarını değiştirme.
- Sadece git sync + .gitignore + git rm --cached + commit + normal push.

Ön kontrol:
1. git fetch origin
2. local main son commit’i göster.
3. origin/main son commit’i göster.
4. local ahead/behind durumunu göster.
5. Eğer local 16 ahead ve 1 behind ise, origin/main’deki commit’i normal merge ile al:
   git merge --no-edit origin/main
   Eğer conflict çıkarsa dur ve raporla; çözmeye çalışma.
