Amaç:

KURALLAR:
- Git history rewrite yapma.
- git filter-repo çalıştırma.
- git filter-branch çalıştırma.
- force push yapma.
- Dosyaları çalışma alanından fiziksel olarak silme.
- DB, schema, migration, package.json, env değiştirme.
