## PART 1: SUPERVISOR MISSION CONTROL

Create `client/src/components/mission-control/SupervisorDashboard.tsx`

### KPIs (6 metrics):
1. Burada (active staff count, green)
2. Geç (late count, amber if > 0)
3. Molada (break count, purple)
4. Gelmedi (absent count, red if > 0)
5. Checklist (completion %, green > 80)
6. Performans (branch score %, green > 80)

### Presence Bar (NEW component — shared/PresenceBar.tsx):
- Row of 32px circles, one per staff member
- Circle shows initials (or photo if available)
- Border color = status: green=active, amber=late, purple=break, red=absent, dashed gray=OFF
- Hover: tooltip with name + shift time
- At end: "+3" overflow circle if > 10 visible
- Legend below: colored dots with labels

### Staff Cards (4-column grid):
Each card (min 48px touch target):
- 40px avatar with 2.5px colored ring (status)
- Name (10px bold, truncated)
- Role (8px muted)
- Shift time or status text
- Score on right (16px, color-coded)
- Trend arrow (↑+3 or ↓-5)

States: active (green ring), late (amber ring), break (purple ring), absent (red ring), off (dashed gray ring)

Data source: use existing branch staff + PDKS data queries from sube-ozet.tsx

### Shift Timeline (Gantt bars — shared/ShiftTimeline.tsx):
- Left column: staff names (55px)
- Right: horizontal time track (07:00-19:00)
- Each person: colored bar showing shift time
  - Green: active/on-time
  - Amber: late start
  - Purple overlay: break period
  - Gray: completed
  - Red text: "Gelmedi" for absent
  - Dashed: OFF
- Time markers at top: 07, 09, 11, 13, 15, 17, 19
- Current time indicator: vertical dashed line

### Checklist Section:
- Each item: checkbox + text + completion time
- Done items: green checkbox with ✓
- Pending items: empty checkbox + estimated time in amber

### Task Feed (same as HQ but branch-filtered):
- Priority bars + title + meta + time
- Data filtered to current user's branch

### Live Activity Feed (branch-specific):
- Same format as HQ but only shows this branch's events

### Mobile (< 768px):
- Presence bar scrolls horizontally
- Staff grid → 2 columns
- Shift timeline → hidden (too complex for mobile), replaced with simple list
- Single column stack

## PART 2: STAJYER/BARISTA MISSION CONTROL

Create `client/src/components/mission-control/StajyerDashboard.tsx`

This is the most DIFFERENT dashboard — education-focused, gamified.

### Mr. Dobody personality: "Kişisel eğitim koçu"
- Feed items focus on: lessons, quizzes, streaks, career tips
- Motivational language: "4 gün seri! Bugün tamamlarsan 5 gün olur!"
- Quiz tips: "Allerjen bölümünde 2 hatan vardı — tekrar et"
- Career nudges: "Bar Buddy'ye 3 modül kaldı"

### KPIs (6 metrics):
1. Eğitim İlerleme (overall %, purple)
2. Gün Serisi (streak count, green)
3. Ort. Quiz Puanı (average, green > 80, amber 60-80, red < 60)
4. Tamamlanan/Toplam (e.g. "5/8")
5. Kalan Modül (count, amber)
6. Sertifika (earned count, green)

### Career Roadmap (shared/CareerRoadmap.tsx):
- Horizontal progression: Stajyer → Bar Buddy → Barista → Sup. Buddy → Supervisor
- Each step: 28px circle with number or ✓
- Done steps: green filled circle
- Current step: purple outlined circle
- Locked steps: gray outlined circle
- Progress line between steps: green fill up to current position
- Below each: level name (9px) + status text (8px muted)

Data: use existing career/level data from academy pages

### Continue Learning Card (prominent, top of content):
- Highlighted card with purple border
- Shows current lesson: module name + lesson title + duration + type
- "Devam et →" button/link
- This is the PRIMARY action — most visible element after KPIs

### Learning Streak (shared/LearningStreak.tsx — Duolingo style):
- 7 day boxes (Mon-Sun): Pt Sa Ça Pe Cu Ct Pz
- Done days: green filled
- Today: purple outlined
- Future: gray
- Missed: red
- Below: motivational text "4 gün seri! Bugün 1 ders = 5 gün!"

### Module List:
Each module card (52px min height):
- Module cover/icon (52px square, colored background)
- Module name (12px bold, truncated)
- Progress text: "3/5 ders" or "Quiz: 88/100"
- Progress bar (4px, purple for active, green for done)
- Badge: "Tamamlandı" (green), "Aktif" (purple), "Zorunlu" (red), "İsteğe bağlı" (blue)
- Active module: purple border highlight

### Quiz Results:
- List of taken quizzes
- Each: icon (✓ green pass, ! amber borderline, ✗ red fail) + name + date + score

### Badge/Rozet Collection (shared/BadgeGrid.tsx):
- Grid of 32px circles
- Earned: colored border, icon/emoji visible
- Locked: dashed gray border, dimmed
- Below each: tiny label (7px)
- Examples: İlk ders ⭐, 5 modül 📚, 3 gün seri 🔥, Tüm quizler 🏆

### Today's Shift (compact):
- Single bar: "09:00-17:00" + "Mola 12:30" badge
- Below: mini checklist of today's tasks (giriş, checklist, mola, çıkış)

### Mobile (< 768px):
- Career roadmap: compact horizontal, smaller circles (20px)
- Continue card: full width, prominent
- Streak: same but smaller boxes (14px)
- Modules: single column list
- Bottom nav: Ana Sayfa | Eğitim (active) | Görevler | Dobody | Profil
