------------------------------------------------------

For EVERY product, the system must store:

- Product name
- Category
- Cup size (Massivo / Long Diva)
- Hot or Iced
- Ingredients (espresso, milk, syrup, powder, sauce, ice, etc.)
- Exact quantities in ml or grams
- Assembly order (step-by-step)
- Photo of final drink
- Common mistakes
- Allergen flags
- Add-ons (extra shot, plant milk, toppings, decaf, lactose-free)

------------------------------------------------------

UI REQUIREMENTS

The main interface must have 3 modes:

1) FLASHCARD MODE
- Big product photo
- Product name
- One-tap to flip
- Front: drink photo + name
- Back: ingredients + steps
- Swipe left/right for next drink
- Difficulty badges (Easy / Medium / Advanced)

2) BARISTA SEARCH MODE
- Search by:
  - Product name
  - Ingredient (e.g. “hazelnut”)
  - Category (e.g. Special Latte)
- Filters:
  - Hot / Iced
  - Coffee / Non-coffee
  - Milk based / Fruit based
- Result cards with quick preview

3) KITCHEN MODE (OPERATIONS)
- No animations
- Big fonts
- Step-by-step brewing mode
- Checkboxes for each step
- Timers for espresso shots, milk steaming, blending, etc.

------------------------------------------------------

TRAINING FEATURES

- Quiz mode: “What goes into a Caramel Macchiato?”
- Error-based training: system shows common mistakes
- Progress tracking per barista
- Certification when a category is mastered

------------------------------------------------------

DATA & ADMIN

HQ must be able to:
- Update recipes centrally
- Push new recipes to all stores
- Lock recipes (so staff cannot change)
- Track which barista used which recipe

------------------------------------------------------

DESIGN STYLE

- Clean, minimal, fast
- High contrast for shop lighting
- Touch friendly
- Works on tablet and kiosk

------------------------------------------------------

The system must behave like:
A mix of:
- Starbucks internal training
- McDonald's kitchen screens
- Duolingo flashcards
- POS speed

Do NOT design a menu for customers.
This is an internal professional barista system.
