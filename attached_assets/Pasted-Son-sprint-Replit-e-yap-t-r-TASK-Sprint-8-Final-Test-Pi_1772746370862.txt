### T004 — Veri Bütünlüğü Final Kontrolü (~30 dakika)

Sprint 7 temizliği sonrası her şey tutarlı mı?

```sql
-- 1. Aktif kullanıcı özeti
SELECT role, COUNT(*) FROM users WHERE is_active = true GROUP BY role ORDER BY count DESC;

-- 2. Aktif şube özeti
SELECT id, name, is_active FROM branches WHERE is_active = true ORDER BY name;

-- 3. Feedback form settings — tüm aktif şubelerde var mı?
SELECT b.name FROM branches b 
LEFT JOIN feedback_form_settings ffs ON ffs.branch_id = b.id 
WHERE b.is_active = true AND ffs.id IS NULL;

-- 4. Kariyer yolu seviyeleri — required_module_ids dolu mu?
SELECT level, title, array_length(required_module_ids, 1) as module_count 
FROM career_path_levels ORDER BY level;

-- 5. Eğitim modülleri — target_roles boş olan var mı?
SELECT COUNT(*) FROM training_modules WHERE target_roles IS NULL OR target_roles = '' OR target_roles = '[]';

-- 6. Gate tanımları aktif mi?
SELECT gate_number, title, is_active FROM career_gates ORDER BY gate_number;

-- 7. Bildirim durumu (temiz mi?)
SELECT 
  COUNT(*) FILTER (WHERE is_archived = false AND is_read = false) as unread_active,
  COUNT(*) FILTER (WHERE is_archived = false AND is_read = true) as read_active,
  COUNT(*) FILTER (WHERE is_archived = true) as archived
FROM notifications;

-- 8. Orphan kontrolü
SELECT COUNT(*) FROM tasks WHERE status IN ('active','pending') AND assigned_to_id IN (SELECT id FROM users WHERE is_active = false);
SELECT COUNT(*) FROM users u JOIN branches b ON u.branch_id = b.id WHERE u.is_active = true AND b.is_active = false;
```

Tüm sorgular tutarlı ve temiz sonuç vermeli.
