### BÖLÜM F: Tüm Sistemde Tekrar Eden Linkler

Bu bölüm en kritik — aynı sayfaya kaç farklı yerden erişiliyor?

```bash
# 1. En çok referans edilen route'lar
grep -rn "to=\"\|to={\|navigate(\|href=\"" client/src/ --include="*.tsx" | \
  grep -oP '(?:to="|to={"|navigate\("|href=")[^"]*' | \
  sort | uniq -c | sort -rn | head -30

# 2. Sidebar'da olan ama dashboard'da da olan linkler
# (Manuel karşılaştırma gerekebilir — sidebar items vs dashboard links)

# 3. Kullanılmayan/kırık route'lar
# Tanımlı ama hiçbir yerden link verilmeyen sayfalar
ls client/src/pages/*.tsx | while read f; do
  page=$(basename "$f" .tsx)
  count=$(grep -rn "$page" client/src/ --include="*.tsx" | grep -v "import\|from\|pages/$page" | wc -l)
  if [ "$count" -lt 2 ]; then
    echo "AZ REFERANS: $page ($count referans)"
  fi
done 2>/dev/null | head -20

# 4. Aynı menü öğesinin farklı label'larla görünmesi
grep -rn "Görevler\|görevler\|Gorevler\|gorevler\|tasks\|Tasks" client/src/ --include="*.tsx" | grep -i "label\|title\|text\|children" | head -10
grep -rn "Vardiya\|vardiya\|shift\|Shift" client/src/ --include="*.tsx" | grep -i "label\|title\|text\|children" | head -10
grep -rn "Raporlar\|raporlar\|report\|Report" client/src/ --include="*.tsx" | grep -i "label\|title\|text\|children" | head -10
```

Raporla:
- En çok tekrar eden linkler hangileri? (aynı sayfaya kaç yerden gidiliyor?)
- Sidebar + Dashboard + Bottom Nav'da aynı anda bulunan öğeler
- Farklı label'larla aynı sayfaya yönlendiren linkler (tutarsızlık)
- Hiçbir yerden link verilmeyen "yetim" sayfalar var mı?
- Kırık linkler var mı? (route tanımlı ama sayfa yok, veya tersi)
