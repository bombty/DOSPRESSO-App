## PART 1: DATABASE — LAYOUT CONFIGURATION

### 1A: dashboard_layouts table (admin templates)

```sql
CREATE TABLE IF NOT EXISTS dashboard_layouts (
  id SERIAL PRIMARY KEY,
  name VARCHAR(100) NOT NULL,
  role_group VARCHAR(50) NOT NULL,  -- 'hq', 'supervisor', 'barista', 'stajyer', 'factory', 'muhasebe', 'custom'
  layout_json JSONB NOT NULL,       -- full widget configuration
  is_default BOOLEAN DEFAULT false,
  created_by VARCHAR(36) REFERENCES users(id),
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);

-- Index for quick lookup
CREATE INDEX IF NOT EXISTS idx_dashboard_layouts_role ON dashboard_layouts(role_group, is_default);
```

### 1B: user_dashboard_prefs table (personal overrides)

```sql
CREATE TABLE IF NOT EXISTS user_dashboard_prefs (
  id SERIAL PRIMARY KEY,
  user_id VARCHAR(36) NOT NULL REFERENCES users(id) ON DELETE CASCADE,
  layout_override_json JSONB,      -- partial override (only changed widgets)
  dobody_expanded BOOLEAN DEFAULT true,
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW(),
  UNIQUE(user_id)
);
```

- Zones: "top" (full width), "main-left" (2/3), "main-right" (1/3)
- Each zone accepts dropped widgets
- Filled slots: show widget preview + name + ⚙ settings + ↑↓ reorder + × remove buttons
- Empty slots: dashed border "Widget sürükle veya tıkla ekle"

### Properties panel (right side):
When a widget is selected, show its configurable options:

**Görev Akışı example:**
- Layout: vertical / horizontal / compact (visual toggles)
- Max görev sayısı: number input (2-10)
- Öncelik çubuğu: toggle on/off
- Tahmini süre: toggle on/off
- "Tümünü gör" linki: toggle on/off
- Atayan kişi fotoğrafı: toggle on/off
- Kart boyutu: select (kompakt/normal/geniş)
- Veri kaynağı: select (tüm görevler/sadece benim/şube görevleri)
- Sıralama: select (öncelik/tarih/süre)
- Arka plan: color picker (transparent/muted/blue-tint/green-tint/amber-tint)
- Çerçeve: select (ince 0.5px/yok/kalın 1px)
- Köşe yuvarlatma: select (keskin 4px/normal 8px/yuvarlak 12px)

**Erişim:**
- Kullanıcı düzenleyebilir: toggle
- Kaldırılabilir: toggle
- Zorunlu widget: toggle

### Drag-drop implementation:
Use @dnd-kit/core (already may be in the project, or install):
```bash
# Check if dnd-kit exists
grep -r "dnd-kit\|@dnd-kit" package.json
# If not, install:
npm install @dnd-kit/core @dnd-kit/sortable @dnd-kit/utilities
```

If installing packages is restricted, implement basic drag-drop with native HTML5 drag events (no library needed):
```typescript
// Each widget item: draggable="true" onDragStart={handleDragStart}

```

### Save flow:
1. Admin edits layout → JSON updated in local state
2. "Önizleme" → opens new tab showing dashboard with this config
3. "Kaydet & Yayınla" → POST/PUT to /api/dashboard/layouts
4. All users of that role group see updated layout on next page load
5. "Sıfırla" → revert to system default

## PART 5: THUMBNAIL MANAGER

### Admin page section or separate page: "Görsel Yönetimi"

Tabs: Şubeler | Eğitim Modülleri | Ürünler | Personel

### Each tab:
- Grid of items (4 columns)
- Each item: current thumbnail (or placeholder) + name + "Fotoğraf yükle" link
- Click → file picker → upload to Replit Object Storage → save URL to DB
- Delete option for existing thumbnail

### Upload flow:
1. User clicks "Fotoğraf yükle"
2. File picker opens (accept: image/jpeg, image/png, image/webp)
3. Client resizes to max 400x400px (for thumbnail use)
4. Upload to existing Replit Object Storage (or /api/upload endpoint)
5. Save URL to appropriate table column:
   - Branches: branches.image_url (add column if not exists)
   - Modules: training_modules.cover_image_url (add column if not exists)  
   - Products: factory_products.image_url (add column if not exists)
   - Users: users.profile_image_url (already exists)

## PART 7: DYNAMIC WIDGET RENDERER

Create `client/src/components/mission-control/WidgetRenderer.tsx`:

```typescript
// Maps widget ID to React component + applies config
const WIDGET_COMPONENTS = {
  'kpi-strip': lazy(() => import('./shared/KPIStrip')),
  'task-feed': lazy(() => import('./shared/TaskFeed')),
  'branch-status': lazy(() => import('./shared/BranchStatus')),
  'live-feed': lazy(() => import('./shared/LiveFeed')),
  'staff-grid': lazy(() => import('./shared/StaffGrid')),
  'shift-timeline': lazy(() => import('./shared/ShiftTimeline')),
  'checklist': lazy(() => import('./shared/ChecklistWidget')),
  'academy-progress': lazy(() => import('./shared/AcademyProgress')),
  'career-roadmap': lazy(() => import('./shared/CareerRoadmap')),
  'factory-summary': lazy(() => import('./shared/FactorySummary')),
  'presence-bar': lazy(() => import('./shared/PresenceBar')),
  'context-bar': lazy(() => import('./shared/ContextBar')),
  'quick-actions': lazy(() => import('./shared/QuickActions')),
  'bordro-summary': lazy(() => import('./shared/BordroSummary')),
  'learning-streak': lazy(() => import('./shared/LearningStreak')),
  'badge-collection': lazy(() => import('./shared/BadgeCollection')),
};

export function WidgetRenderer({ widgetConfig }: { widgetConfig: WidgetConfig }) {
  const Component = WIDGET_COMPONENTS[widgetConfig.id];
  if (!Component || !widgetConfig.enabled) return null;
  
  return (
    <Suspense fallback={<WidgetSkeleton />}>
      <Component config={widgetConfig.config} />
    </Suspense>
  );
}
```

Each dashboard then becomes:
```typescript
function MissionControlDashboard({ layoutConfig }) {
  const topWidgets = layoutConfig.widgets.filter(w => w.zone === 'top').sort(byPosition);
  const leftWidgets = layoutConfig.widgets.filter(w => w.zone === 'main-left').sort(byPosition);
  const rightWidgets = layoutConfig.widgets.filter(w => w.zone === 'main-right').sort(byPosition);
  
  return (
    <div className="flex-1 overflow-y-auto">
      {topWidgets.map(w => <WidgetRenderer key={w.id} widgetConfig={w} />)}
      <div className="grid grid-cols-3 gap-3">
        <div className="col-span-2 space-y-3">
          {leftWidgets.map(w => <WidgetRenderer key={w.id} widgetConfig={w} />)}
        </div>
        <div className="space-y-3">
          {rightWidgets.map(w => <WidgetRenderer key={w.id} widgetConfig={w} />)}
        </div>
      </div>
    </div>
  );
}
```
