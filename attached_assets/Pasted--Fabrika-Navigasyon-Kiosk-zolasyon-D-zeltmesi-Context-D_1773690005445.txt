# Fabrika Navigasyon + Kiosk İzolasyon Düzeltmesi

## Context
DOSPRESSO franchise platform. Stack: React 18 + TypeScript, Wouter routing.

Two separate systems need to be clearly separated:
1. **Fabrika Modülü** — HQ management view (dashboard, analytics, planning)
2. **Kiosk Sistemi** — Isolated PIN-only terminal for factory/branch floor workers

Current problems:
- /fabrika/dashboard is standalone (no nav = HQ users get lost)
- NavRail points to standalone dashboard instead of mega module
- Kiosk has no route guard — anyone with URL can access
- fabrika_operator can see kiosk nav link (they shouldn't navigate there manually)

Do NOT touch: kiosk PIN auth logic, kiosk worker flow, CRM, academy modules.

Add a minimal guard that redirects to login if not authenticated:

```typescript
// Kiosk pages should:
// 1. Still be standalone (no main nav/header)
// 2. But require authentication OR have their own PIN system
// 3. Regular branch/HQ users who somehow reach the URL should be redirected

// Option A: Keep kiosk as public (PIN is the auth) but add a check:
// If user is authenticated as a non-factory/non-admin role, redirect away
// This prevents accidental access by wrong users

// Check how existing auth guards work in App.tsx:
// grep -n "ProtectedRoute\|AuthGuard\|isAuthenticated" client/src/App.tsx | head -10
// Then apply same pattern to kiosk routes
```

For kiosk pages specifically:
- Unauthenticated users → show kiosk PIN login (current behavior, keep it)
- Authenticated non-kiosk users (e.g. coach, muhasebe) → redirect to their dashboard
- Factory floor users and admins → allow through to kiosk

```typescript
// Add kiosk access check component:
function KioskGuard({ children }: { children: React.ReactNode }) {
  const { user } = useAuth();
  
  // Not logged in = show kiosk (PIN login handles auth)
  if (!user) return <>{children}</>;
  
  // Admin can always access kiosk
  if (['admin', 'ceo'].includes(user.role)) return <>{children}</>;
  
  // Factory/branch floor roles can access kiosk
  const kioskRoles = ['fabrika_operator', 'fabrika_sorumlu', 'fabrika_personel', 
                       'fabrika_mudur', 'mudur', 'supervisor'];
  if (kioskRoles.includes(user.role)) return <>{children}</>;
  
  // Everyone else → redirect to their home
  return <Redirect to="/" />;
}
```
