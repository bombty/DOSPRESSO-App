## GENEL PATTERN (4 MODÜL İÇİN AYNI)

Her modül için uygulanan ortak pattern:

```
Modül Sayfası (yeniden yazılmış veya wrapper)
├── Header: Modül adı + açıklama
├── KPI Strip (4 metrik — modüle özel)
├── Sidebar (sol 200px)
│   ├── Genel bölüm: Dashboard, Tümü, [modüle özel filtreler]
│   ├── Kategori bölüm: [modüle özel kategoriler]
│   └── Ayarlar bölüm: [modüle özel ayarlar]
├── Ana alan
│   ├── İçerik listesi (sidebar seçimine göre filtrelenir)
│   └── Trend/özet grafik (alt bölüm — opsiyonel)
└── Detay panel (tıklanınca sağ panel veya modal)
```

**⚠ ÖNEMLİ — Mevcut kodu silme, yeniden kullan:**
- Mevcut tab içerikleri = yeni sidebar görünümlerinin içeriği
- Tab1'in içeriği → sidebar "Tümü" seçildiğinde gösterilir
- Tab2'nin içeriği → sidebar "Kategori X" seçildiğinde gösterilir
- Mevcut query'ler, mevcut bileşenler aynen kullanılır
- Sadece LAYOUT değişiyor — veri akışı aynı

## MODÜL 1: AKADEMİ

### Mevcut durum (ön araştırmadan belirlenecek):
- Muhtemelen: Modüller, Quizler, Sertifikalar, İstatistikler, Onboarding tab'ları

### Yeni yapı:

**KPI Strip:**
1. Aktif öğrenci sayısı (eğitim alan)
2. Tamamlama oranı (tüm modüller ort. %)
3. Ortalama quiz puanı
4. Bu ay verilen sertifika sayısı

**Sidebar:**
```
GENEL
  ● Dashboard (KPI + özet)
  ● Tüm modüller
  ● Quiz sonuçları
  ● Sertifikalar

KATEGORİLER
  ○ Zorunlu eğitimler
  ○ Barista eğitimleri
  ○ Gıda güvenliği
  ○ Liderlik
  ○ Ürün bilgisi

YÖNETİM
  ○ Onboarding
  ○ İçerik paketleri
  ○ Webinarlar
  ○ Ayarlar
```

**Ana alan:**
- Dashboard seçili → KPI + son tamamlamalar listesi + haftalık trend grafik
- Tüm modüller seçili → modül kartları grid (mevcut modül listesi bileşeni)
- Quiz sonuçları seçili → quiz listesi (mevcut quiz bileşeni)
- Sertifikalar seçili → sertifika listesi
- Kategori seçili → o kategorinin modülleri filtrelenmiş

**Detay:** Modül tıklanınca → mevcut modül detay sayfası açılır (veya sağ panel)

### Mevcut durum (ön araştırmadan belirlenecek):
- Muhtemelen: Personel listesi, Özlük dosyası, Tutanaklar, İzinler, Devam takibi, Bordro

### Yeni yapı:

**KPI Strip:**
1. Aktif personel sayısı

3. Açık tutanak sayısı
4. Bu ay izin kullanım oranı

**Sidebar:**
```

  ● Tüm personel (263 aktif)

MODÜLLER
  ○ Özlük dosyaları
  ○ Tutanaklar
  ○ İzin yönetimi
  ○ Devam takibi (PDKS)
  ○ Vardiya planı
  ○ Bordro

  ○ Şubeye göre (dropdown)
  ○ Role göre (dropdown)
```

**Ana alan:**
- Dashboard → KPI + son işlemler + yaklaşan izinler + süresi dolan belgeler

- Sadece ana İK dashboard'u oluştur (KPI + sidebar navigasyon)
- Sidebar'daki öğeler tıklanınca mevcut sayfalara navigate etsin (/personel, /vardiyalar vb.)
- Tab'ları öldürmek yerine, İK hub sayfası + mevcut alt sayfalar yaklaşımı

## UYGULAMA STRATEJİSİ

### Her modül için adımlar:

1. **Araştır:** Mevcut sayfa yapısını tam anla (tab'lar, query'ler, bileşenler)
2. **Sidebar bileşeni oluştur:** `client/src/components/[modul]/[Modul]Sidebar.tsx`
3. **Dashboard bileşeni oluştur:** `client/src/components/[modul]/[Modul]Dashboard.tsx` (KPI + özet)
4. **Ana sayfa wrapper yaz:** Mevcut sayfa kodunu sidebar + content layout'a sar
5. **Tab mantığını sidebar mantığına çevir:** useState('activeTab') → useState('activeView')
6. **Test:** Tüm mevcut özellikler çalışıyor mu?

### Sidebar bileşeni pattern (tüm modüller için aynı):

```typescript
interface SidebarProps {
  activeView: string;
  onViewChange: (view: string) => void;
  badges?: Record<string, number>; // okunmamış/açık sayıları
}

function ModulSidebar({ activeView, onViewChange, badges }: SidebarProps) {
  return (
    <div className="w-[200px] border-r p-3 space-y-4">
      <SidebarSection title="GENEL">
        <SidebarItem active={activeView === 'dashboard'} onClick={() => onViewChange('dashboard')} label="Dashboard" />
        <SidebarItem active={activeView === 'all'} onClick={() => onViewChange('all')} label="Tümü" badge={badges?.all} />
      </SidebarSection>
      {/* modüle özel bölümler */}
    </div>
  );
}
```

### KPI strip pattern:

```typescript
function KPIStrip({ metrics }: { metrics: KPIMetric[] }) {
  return (
    <div className="grid grid-cols-4 gap-3 mb-4">
      {metrics.map(m => (
        <div key={m.label} className="bg-muted/30 rounded-lg p-3 text-center">
          <div className={`text-xl font-medium ${m.color}`}>{m.value}</div>
          <div className="text-xs text-muted-foreground mt-1">{m.label}</div>
        </div>
      ))}
    </div>
  );
}
```

## MOBİL RESPONSIVE

Her modül için mobil (<768px) düzeni:

- Sidebar → üst yatay scroll tab bar'a dönüşür (compact pill butonlar)
- KPI strip → 2x2 grid
- İçerik listesi → tek kolon, full width
- Detay panel → tam ekran modal

```typescript
// Responsive sidebar → mobile tab bar
const isMobile = useMediaQuery('(max-width: 768px)');

if (isMobile) {
  return (
    <div className="flex gap-2 overflow-x-auto p-2 border-b">
      {items.map(item => (
        <button key={item.id} className={cn("whitespace-nowrap px-3 py-1.5 rounded-full text-xs", 
          active === item.id ? "bg-primary text-primary-foreground" : "bg-muted"
        )} onClick={() => onViewChange(item.id)}>
          {item.label} {item.badge && <span className="ml-1 text-[10px]">{item.badge}</span>}
        </button>
      ))}
    </div>
  );
}
```
