### T008 — Yazı Boyutu & Input Zoom Fix (~1 saat)

**Yapılacak:**

```bash
# text-xs kullanımları — telefonda çok küçük olabilir
grep -rn "text-xs" client/src/ --include="*.tsx" | wc -l
```

Kurallar:
- Ana metin: minimum text-sm (14px)
- Etiket/badge: text-xs (12px) kabul edilebilir
- Buton metni: minimum text-sm (14px)
- Input: minimum text-base (16px) — **16px altı Safari'de otomatik zoom tetikler!**
- Başlık: minimum text-lg (18px)

**Safari zoom sorunu kritik:**
```bash
grep -rn "text-sm\|text-xs" client/src/components/ui/input* --include="*.tsx" | head -5
```
Eğer input font-size < 16px ise `text-base` yap. Aksi halde her input'a dokunduğunda sayfa zoom yapar.
