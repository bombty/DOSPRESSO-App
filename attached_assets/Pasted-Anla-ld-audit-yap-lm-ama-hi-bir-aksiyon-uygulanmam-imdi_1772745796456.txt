### T002 — Orphan Görevleri Temizle (~30 dakika)

149 orphan görev (pasif kullanıcılara atanmış, hâlâ active/pending statüsünde).

```sql
-- 1. Önce listele
SELECT t.id, t.title, t.status, t.assigned_to_id, u.username, u.is_active 
FROM tasks t 
JOIN users u ON t.assigned_to_id = u.id 
WHERE t.status IN ('active', 'pending') AND u.is_active = false
LIMIT 10;

-- 2. Pasif kullanıcıya atanmış görevleri "cancelled" yap
UPDATE tasks SET status = 'cancelled', updated_at = NOW() 
WHERE status IN ('active', 'pending') AND assigned_to_id IN (
  SELECT id FROM users WHERE is_active = false
);

-- 3. assigned_to_id NULL olan aktif görevleri de iptal et
UPDATE tasks SET status = 'cancelled', updated_at = NOW() 
WHERE status IN ('active', 'pending') AND assigned_to_id IS NULL;
```

**Doğrulama:**
```sql
SELECT COUNT(*) FROM tasks WHERE status IN ('active', 'pending') AND assigned_to_id IN (SELECT id FROM users WHERE is_active = false);
-- Hedef: 0
```

### T004 — Eski Bildirimleri Arşivle (~30 dakika)

16.773 arşivlenmemiş bildirim + 18.377 reminder kaydı.

```sql
-- 1. 30 günden eski arşivlenmemiş bildirimleri arşivle
UPDATE notifications SET is_archived = true 
WHERE is_archived = false AND created_at < NOW() - INTERVAL '30 days';

-- 2. Okunmuş ama arşivlenmemiş bildirimleri de arşivle (7 günden eski)
UPDATE notifications SET is_archived = true 
WHERE is_archived = false AND is_read = true AND created_at < NOW() - INTERVAL '7 days';

-- 3. Sonucu kaydet
SELECT 
  COUNT(*) FILTER (WHERE is_archived = true) as archived,
  COUNT(*) FILTER (WHERE is_archived = false) as active
FROM notifications;
```
