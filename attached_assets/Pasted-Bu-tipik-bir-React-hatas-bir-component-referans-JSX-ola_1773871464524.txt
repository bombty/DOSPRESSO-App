## CRITICAL FIX: Academy HQ React Runtime Error

### Error
"Objects are not valid as a React child (found: object with keys {$$typeof, render})"
This happens when Coach (Yavuz) navigates to /akademi-hq.

### Root Cause Investigation
This error means a React component REFERENCE is being passed as a child instead of being RENDERED as JSX. For example:
- WRONG: {MyComponent} or {SomeIcon}
- CORRECT: <MyComponent /> or <SomeIcon />

```bash
# 1. Check akademi-hq/index.tsx for component references used as children
cat client/src/pages/akademi-hq/index.tsx

# 2. Check all tab imports — are they default exports or named exports?
grep -n "import\|export default\|export function" client/src/pages/akademi-hq/*.tsx | head -30

# 3. Check for Lucide icons used incorrectly (common cause)
grep -n "icon\|Icon\|{.*Icon}" client/src/pages/akademi-hq/index.tsx | head -20

# 4. Check RoleDashboard component
cat client/src/pages/akademi-hq/components/RoleDashboard.tsx | head -60

# 5. Check if any tab component returns a forwardRef or memo wrapper incorrectly
grep -n "forwardRef\|memo\|React.memo\|React.forwardRef" client/src/pages/akademi-hq/*.tsx | head -10
```

### Common patterns that cause this error:

Pattern 1 — Icon as child:
```typescript
// WRONG:
<Badge>{SomeIcon} Aktif</Badge>
// CORRECT:
<Badge><SomeIcon className="h-4 w-4" /> Aktif</Badge>
```

Pattern 2 — Component reference instead of render:
```typescript
// WRONG:
{ModullerTab}
// CORRECT:
<ModullerTab />
```

Pattern 3 — Shadcn TabsContent with component reference:
```typescript
// WRONG:
<TabsContent value="moduller">{ModullerTab}</TabsContent>
// CORRECT:
<TabsContent value="moduller"><ModullerTab /></TabsContent>
```

Pattern 4 — Array.map returning component reference:
```typescript
// WRONG:
tabs.map(tab => tab.component)
// CORRECT:
tabs.map(tab => <tab.component />)
// OR:
tabs.map(tab => { const Comp = tab.component; return <Comp />; })
```

### Fix
Find and fix ALL instances of the pattern in akademi-hq files. The fix is usually adding < /> around component references.
