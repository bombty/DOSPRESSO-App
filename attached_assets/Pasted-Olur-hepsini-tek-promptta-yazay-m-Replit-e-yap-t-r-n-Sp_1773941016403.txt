```typescript
router.get("/api/certificates/my", isAuthenticated, async (req, res) => {
  const user = req.user as any;
  const certs = await db
    .select()
    .from(issuedCertificates)
    .where(and(
      eq(issuedCertificates.recipientUserId, user.id),
      eq(issuedCertificates.isActive, true),
```

### Implementation — PIN-First Login

Instead of showing all users, require PIN entry first:

#### 3A: Change login flow

**Old flow:** Device Password → Select User (shows all names) → Enter PIN
**New flow:** Device Password → Enter PIN (numeric keypad only) → System identifies user from PIN

```typescript
// The PIN is unique per user per branch/factory
// When user enters PIN, system looks up which user has this PIN
// If found → login successful
// If not found → "Geçersiz PIN" error

// This way, no personnel list is ever shown on screen
```

#### 3B: Check if PINs are unique per branch

```bash
# How are kiosk PINs stored?

# Are PINs unique?
```

If PINs are NOT unique per branch, this approach won't work. Alternative: show initials only (not full names), or use employee ID number entry.
