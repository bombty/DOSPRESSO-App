═══ İSTEDİĞİM ADIMLAR ═══
1. Önce mevcut durumu kontrol et: git log --oneline -1 ile hangi commit'te olduğumuzu göster
2. GitHub'dan son kodu zorla çek:
   git fetch origin main
   git reset --hard origin/main
3. Eski build'i temizle: rm -rf dist
4. Yeniden build et: npm run build — sonucu göster
5. Sunucuyu yeniden başlat
6. Sonra şu kullanıcılarla test et:
   - Admin/CEO kullanıcı → CEO dashboard açılmalı
   - Yavuz (Coach) → Coach dashboard 5 sekmeli açılmalı
   - Ece (Trainer) → Trainer dashboard 5 sekmeli açılmalı
   - Utku (CGO) → CGO dashboard açılmalı
7. Her dashboard'da kontrol et:
   - CentrumShell layout var mı? (üstte KPI strip, sağda Dobody panel)
   - Widget'lar grid'de mi?
   - Koyu tema renkleri mi kullanılıyor? (#0c0f14 arka plan, #141820 kartlar)

Eğer hâlâ eski tasarım görünüyorsa:
- Browser cache sorunu olabilir → dist klasörünü silip tekrar build et
- Vite dev server cache → node_modules/.vite silmeyi dene

ÖNEMLİ: Hiçbir dosyayı değiştirme, sadece GitHub'dan çek ve build et. Tüm kod değişiklikleri IT danışmanı Claude tarafından yapılıyor.
