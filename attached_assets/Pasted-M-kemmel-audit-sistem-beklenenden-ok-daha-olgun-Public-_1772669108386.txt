**B) Composite Score'a Entegrasyon:**

Mevcut Composite Score'un "Practical" bileşenine (%25) müşteri geri bildirim etkisini ekle:

Mevcut Practical hesaplamayı bul:
```bash
grep -rn "practical\|Practical\|checklist.*score\|task.*score" server/ --include="*.ts" | head -15
```

Yeni Practical hesaplama:
- Checklist tamamlama: %40 ağırlık (mevcut)
- Görev zamanında bitirilme: %30 ağırlık (mevcut)
- **Müşteri geri bildirim etkisi: %30 ağırlık (YENİ)**

Müşteri geri bildirim etkisi hesaplama:

**Temizlik puanı (supervisor/supervisor_buddy için):**
```sql
-- Son 30 günde şubenin temizlik ortalaması
```

- 4.0+ → +10 puan (bonus)
- 3.5-4.0 → 0 (nötr)
- 3.0-3.5 → -5 puan
- <3.0 → -10 puan

**Güler yüzlülük (bireysel personel için):**
```sql

```
- Minimum 3 değerlendirme biriktikten sonra aktif olur
- 4.0+ → +10 puan (bonus)
- 3.5-4.0 → 0 (nötr)
- <3.5 → -8 puan
- Bu etki SADECE ismi verilen personelin skoruna yansır

**Ürün kalitesi (şube genel):**
```sql
SELECT AVG(product_rating) FROM customer_feedback 
WHERE branch_id = [şube] AND created_at > NOW() - INTERVAL '30 days';
```
- Tüm şube personeline hafif etki: ±3 puan max
