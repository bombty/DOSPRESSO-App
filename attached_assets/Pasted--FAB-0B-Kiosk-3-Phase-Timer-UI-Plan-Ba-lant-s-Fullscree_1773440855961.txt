## BACKGROUND: 3 Problems Being Fixed

**BF-01:** Kiosk fullscreen button throws "Cannot request fullscreen without transient activation" error.
**BF-02:** Kiosk shows "Bugün için atanmış üretim planı yok" even when a plan exists in the DB.

## TASK 1 — Fix Fullscreen Error (BF-01)

FILE: The kiosk component that has the fullscreen button.

The error "Cannot request fullscreen without transient activation" happens when 
`document.documentElement.requestFullscreen()` is called programmatically on page load 
or from a non-user-gesture context.

FIX: Fullscreen must ONLY be triggered by a direct user click event.
- Remove any automatic fullscreen calls (on mount, on timer start, etc.)
- The fullscreen button click handler must call `requestFullscreen()` synchronously 
  within the click event — no setTimeout, no async/await before the call
- Add error handling:

```typescript
const handleFullscreen = () => {
  if (!document.fullscreenElement) {
    document.documentElement.requestFullscreen().catch((err) => {
      console.warn("Fullscreen request failed:", err);
    });
  } else {
    document.exitFullscreen().catch((err) => {
      console.warn("Exit fullscreen failed:", err);
    });
  }
};
```

### 3B. Backend — Phase transition endpoint

In `server/routes/factory.ts`, add a new endpoint:

```
PATCH /api/factory/kiosk/session/:sessionId/phase
```

Body: `{ phase: "hazirlik" | "uretim" | "temizlik" | "tamamlandi" }`

Logic:
```typescript
router.patch("/kiosk/session/:sessionId/phase", isAuthenticated, async (req, res) => {
  const { sessionId } = req.params;
  const { phase } = req.body;
  const now = new Date();

  // Get current session
  const session = await db.query.factoryShiftSessions.findFirst({
    where: eq(factoryShiftSessions.id, parseInt(sessionId))
  });
  if (!session) return res.status(404).json({ error: "Session not found" });

  const updates: Record<string, any> = { phase };

  if (phase === "uretim") {
    updates.prodStartedAt = now;
    // Calculate prep duration
    if (session.prepStartedAt) {
      updates.prepDurationMinutes = Math.floor(
        (now.getTime() - new Date(session.prepStartedAt).getTime()) / 60000
      );
    }
  } else if (phase === "temizlik") {
    updates.cleanStartedAt = now;
    updates.prodEndedAt = now;
    if (session.prodStartedAt) {
      updates.prodDurationMinutes = Math.floor(
        (now.getTime() - new Date(session.prodStartedAt).getTime()) / 60000
      );
    }
  } else if (phase === "tamamlandi") {
    updates.endTime = now; // or whatever the end column is called
    if (session.cleanStartedAt) {
      updates.cleanDurationMinutes = Math.floor(
        (now.getTime() - new Date(session.cleanStartedAt).getTime()) / 60000
      );
    }
  }

  await db.update(factoryShiftSessions)
    .set(updates)
    .where(eq(factoryShiftSessions.id, parseInt(sessionId)));

  res.json({ success: true, phase, updatedAt: now });
});
```

Also add `prep_started_at = NOW()` when a session is first created (when worker starts their shift at a station).
