### PART 1: Create Reusable CompactKPIStrip Component

Create `client/src/components/compact-kpi-strip.tsx`:

This component replaces large KPI card grids on mobile with a horizontal scrollable strip:

```typescript
interface KPIItem {
  label: string;
  value: number | string;
  icon?: React.ReactNode;
  color?: "default" | "danger" | "warning" | "success" | "info";
  onClick?: () => void;
}

interface CompactKPIStripProps {
  items: KPIItem[];
  desktopColumns?: 2 | 3 | 4 | 5;
  className?: string;
}
```

**Mobile layout (< md):**
- Horizontal scrollable strip (overflow-x-auto, flex, gap-2, snap-x)
- Each item: min-w-[100px], h-12, rounded-lg, flex items-center gap-2, px-3
- Label: text-xs text-muted-foreground
- Value: text-sm font-semibold
- Color dot/border-left for status (danger=red, warning=amber, success=green)
- Scrollbar hidden (scrollbar-hide)
- If item has onClick, show subtle tap indicator

**Desktop layout (>= md):**
- Standard grid (grid-cols-{desktopColumns}) with the existing card style
- Each item: Card with icon, label, value — same as current design
- Heights stay as they are on desktop
