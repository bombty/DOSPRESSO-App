## ÖN ARAŞTIRMA

```bash
echo "════════════════════════════════════════"
echo "MEVCUT MENÜ DURUMU — ÖNCESİ SNAPSHOT"
echo "════════════════════════════════════════"

echo "=== Mevcut bölümler ve öğeler ==="
# Admin login
curl -s -X POST http://localhost:5000/api/login \
  -H "Content-Type: application/json" \
  -d '{"username":"utku","password":"0000"}' \
  -c /tmp/menu_test.txt > /dev/null 2>&1

curl -s http://localhost:5000/api/me/menu -b /tmp/menu_test.txt | node -e "
const d=JSON.parse(require('fs').readFileSync('/dev/stdin','utf8'));
let total=0;
d.forEach(s=>{
  const count = (s.items||[]).length;
  total += count;
  console.log('['+s.label+'] ('+count+' öğe)');
  if(s.items) s.items.forEach(i=>console.log('  '+i.id+' → '+i.label+' → '+(i.path||'')));
});
console.log('\nTOPLAM: '+total);
"

echo ""
echo "=== Tüm roller menü sayısı (öncesi) ==="
for ROLE in admin ceo cgo coach trainer muhasebe_ik fabrika_mudur supervisor mudur barista stajyer yatirimci_branch gida_muhendisi satinalma marketing kalite_kontrol; do
  USER=$(echo "SELECT username FROM users WHERE role='$ROLE' AND is_active=true LIMIT 1;" | psql -t 2>/dev/null | tr -d ' ')
  if [ -n "$USER" ]; then
    curl -s -X POST http://localhost:5000/api/login \
      -H "Content-Type: application/json" \
      -d "{\"username\":\"$USER\",\"password\":\"0000\"}" \
      -c /tmp/r_${ROLE}.txt > /dev/null 2>&1
    COUNT=$(curl -s http://localhost:5000/api/me/menu -b /tmp/r_${ROLE}.txt | node -e "
      const d=JSON.parse(require('fs').readFileSync('/dev/stdin','utf8'));
      let c=0; d.forEach(s=>{if(s.items)c+=s.items.length}); console.log(c);
    " 2>/dev/null)
    echo "  $ROLE: $COUNT"
  fi
done

echo ""
echo "=== menu-service.ts bölüm yapısı ==="
grep -n "label.*:\|section\|category\|group" server/menu-service.ts | grep -i "operasyon\|ik\|eğitim\|egitim\|denetim\|finans\|crm\|ileti\|yönetim\|yonetim\|ana.sayfa\|pazarlama\|destek" | head -20

echo ""
echo "=== PDKS ve Maaş şu an hangi bölümde? ==="
grep -n "pdks\|maas\|salary\|payroll" server/menu-service.ts | head -10

echo ""
echo "=== AI Asistan ve Agent Merkezi şu an hangi bölümde? ==="
grep -n "ai-asistan\|ai_asistan\|agent-center\|agent_center\|agent.*merkez" server/menu-service.ts | head -10

echo ""
echo "=== Duplikat sayfa dosyaları ==="
echo "--- kullanicilar ---"
find client/src/pages -name "*kullanicilar*" | sort
echo "--- duyurular/announcements ---"
find client/src/pages -name "*duyuru*" -o -name "*announce*" | sort
echo "--- destek ---"
find client/src/pages -name "*destek*" -o -name "*support*" | sort
```

## ADIM 1: MENÜ ÖĞESİ TAŞIMALARI (menu-service.ts)

Aşağıdaki taşımaları `server/menu-service.ts`'te yap:

### 1A: PDKS + Maaş Hesaplama → İK & Vardiya bölümüne taşı

```
ÖNCEKİ:
  [Finans & Tedarik] → Muhasebe, Mali Yönetim, Satınalma, Stok, PDKS, Maaş Hesaplama (6 öğe)
  [İK & Vardiya] → Personel, Vardiyalar, Devam Takibi (3 öğe)

SONRAKI:
  [Finans & Tedarik] → Muhasebe, Mali Yönetim, Satınalma, Stok (4 öğe)
  [İK & Personel] → Personel, Vardiyalar, Devam Takibi, PDKS, Maaş Hesaplama (5 öğe)
```

**Bölüm adını da değiştir:** "İK & Vardiya" → "İK & Personel"

### 1B: AI Asistan → Eğitim bölümüne taşı

```
ÖNCEKİ:
  [İletişim & Destek] → Destek Merkezi, Bildirimler, AI Asistan, Agent Merkezi (4 öğe)
  [Eğitim] → Akademi Yönetimi, Bilgi Bankası (2 öğe)

SONRAKI:
  [İletişim] → Bildirimler, Destek Merkezi (2 öğe)
  [Eğitim] → Akademi Yönetimi, Bilgi Bankası, AI Asistan (3 öğe)
```

### 1C: Agent Merkezi → Yönetim bölümüne taşı

```
ÖNCEKİ:
  [İletişim & Destek] → ... Agent Merkezi ...

SONRAKI:
  [Yönetim] → Admin Paneli, Kullanıcılar, Şubeler, ..., Agent Merkezi, ... (sadece admin/ceo/cgo görür)
```

Agent Merkezi sadece admin, ceo, cgo rollerinin görmesi gereken bir yönetim aracı — İletişim'de olmamalı.

### 1D: Misafir Memnuniyeti → CRM bölümüne taşı (opsiyonel)

```
ÖNCEKİ:
  [Denetim & Analitik] → Performans, Raporlar, Kalite, Misafir Memnuniyeti, Gıda Güvenliği (5 öğe)
  [CRM & Pazarlama] → CRM, Yatırımcılar, Kampanyalar (3 öğe)

SONRAKI:
  [Denetim & Analitik] → Performans, Raporlar, Kalite, Gıda Güvenliği (4 öğe)
  [Müşteri İlişkileri] → CRM, Yatırımcılar, Kampanyalar, Misafir Memnuniyeti (4 öğe)
```

**Bölüm adını da değiştir:** "CRM & Pazarlama" → "Müşteri İlişkileri"

### 1E: Bölüm adı güncelleme

| Eski | Yeni |
|------|------|
| İK & Vardiya | İK & Personel |
| CRM & Pazarlama | Müşteri İlişkileri |
| İletişim & Destek | İletişim |

### 1F: SIDEBAR_ALLOWED_ITEMS güncelle

Taşınan öğeler için TÜM rollerin whitelist'ini güncelle:

- `pdks` ve `maas-hesaplama` (veya ID'si ne ise) → İK bölümüne taşındığı için, bu öğeleri gören rollerin listesinde doğru bölümde olduğundan emin ol
- `ai-asistan` → Eğitim bölümüne taşındığı için, eğitim görebilen roller AI Asistan'ı da görmeli
- `agent-center` → sadece admin, ceo, cgo'nun Yönetim bölümünde görmesi gerekiyor

**⚠ DİKKAT:** Öğelerin hangi bölümde göründüğü SIDEBAR_ALLOWED_ITEMS'taki sıralamayla değil, bölüm tanımıyla belirleniyor olabilir. menu-service.ts'in yapısını incele — bölüm ataması nasıl çalışıyor?

## ADIM 2: ROL BAZLI İYİLEŞTİRME

### 2A: Yatırımcı → Finansal raporlama ekle

Yatırımcı şu an sadece temel sayfaları görüyor. Finansal özet raporlarını (raporlar veya mali-yonetim) görmeli:
```
yatirimci_branch whitelist'ine ekle: raporlar (veya reports ID'si)
```

### 2B: CEO → Akademi Yönetimi ekle

CEO stratejik bakış açısıyla eğitim durumunu görmeli:
```
ceo whitelist'ine ekle: akademi-yonetimi (veya academy-admin ID'si)
```

### 2C: Fabrika müdür → Performans ekle

Fabrika müdür Raporlar görüyor ama Performans yok — ikisi birlikte anlamlı:
```
fabrika_mudur whitelist'ine ekle: performans (veya performance ID'si)
```

## ADIM 3: DOĞRULAMA

```bash
echo "════════════════════════════════════════"
echo "MENÜ KONSOLİDASYONU — SONRASI SNAPSHOT"

echo "=== Yeni bölümler ve öğeler ==="
curl -s -X POST http://localhost:5000/api/login \
  -H "Content-Type: application/json" \
  -d '{"username":"utku","password":"0000"}' \
  -c /tmp/menu_after.txt > /dev/null 2>&1

curl -s http://localhost:5000/api/me/menu -b /tmp/menu_after.txt | node -e "
const d=JSON.parse(require('fs').readFileSync('/dev/stdin','utf8'));
let total=0;
d.forEach(s=>{
  const count = (s.items||[]).length;
  total += count;
  console.log('['+s.label+'] ('+count+' öğe)');
  if(s.items) s.items.forEach(i=>console.log('  '+i.id+' → '+i.label));
});
console.log('\nTOPLAM: '+total);
"

echo ""
echo "=== Tüm roller menü sayısı (sonrası) ==="
for ROLE in admin ceo cgo coach trainer muhasebe_ik fabrika_mudur supervisor mudur barista stajyer yatirimci_branch gida_muhendisi satinalma marketing kalite_kontrol; do
  USER=$(echo "SELECT username FROM users WHERE role='$ROLE' AND is_active=true LIMIT 1;" | psql -t 2>/dev/null | tr -d ' ')
  if [ -n "$USER" ]; then
    curl -s -X POST http://localhost:5000/api/login \
      -H "Content-Type: application/json" \
      -d "{\"username\":\"$USER\",\"password\":\"0000\"}" \
      -c /tmp/r2_${ROLE}.txt > /dev/null 2>&1
    COUNT=$(curl -s http://localhost:5000/api/me/menu -b /tmp/r2_${ROLE}.txt | node -e "
      const d=JSON.parse(require('fs').readFileSync('/dev/stdin','utf8'));
      let c=0; d.forEach(s=>{if(s.items)c+=s.items.length}); console.log(c);
    " 2>/dev/null)
    echo "  $ROLE: $COUNT"
  fi
done

echo ""
echo "=== Kontrol listesi ==="
MENU=$(curl -s http://localhost:5000/api/me/menu -b /tmp/menu_after.txt)

echo "--- PDKS İK bölümünde mi? ---"
echo "$MENU" | node -e "
const d=JSON.parse(require('fs').readFileSync('/dev/stdin','utf8'));
d.forEach(s=>{
  if(s.items) s.items.forEach(i=>{
    if(i.id && (i.id.includes('pdks') || i.label && i.label.includes('PDKS')))
      console.log('PDKS → ['+s.label+'] '+(s.label.includes('İK')||s.label.includes('IK')?'✅ DOĞRU':'❌ YANLIŞ'));
    if(i.id && (i.id.includes('maas') || i.id.includes('salary') || (i.label && i.label.includes('Maaş'))))
      console.log('Maaş → ['+s.label+'] '+(s.label.includes('İK')||s.label.includes('IK')?'✅ DOĞRU':'❌ YANLIŞ'));
  });
});
"

echo "--- AI Asistan Eğitim bölümünde mi? ---"
echo "$MENU" | node -e "
const d=JSON.parse(require('fs').readFileSync('/dev/stdin','utf8'));
d.forEach(s=>{
  if(s.items) s.items.forEach(i=>{
    if(i.id && (i.id.includes('ai') || (i.label && i.label.includes('AI'))))
      console.log('AI Asistan → ['+s.label+'] '+(s.label.toLowerCase().includes('eğitim')||s.label.toLowerCase().includes('egitim')?'✅ DOĞRU':'❌ YANLIŞ'));
  });
});
"

echo "--- Agent Merkezi Yönetim bölümünde mi? ---"
echo "$MENU" | node -e "
const d=JSON.parse(require('fs').readFileSync('/dev/stdin','utf8'));
d.forEach(s=>{
  if(s.items) s.items.forEach(i=>{
    if(i.id && (i.id.includes('agent') || (i.label && i.label.includes('Agent'))))
      console.log('Agent → ['+s.label+'] '+(s.label.toLowerCase().includes('yönetim')||s.label.toLowerCase().includes('yonetim')?'✅ DOĞRU':'❌ YANLIŞ'));
  });
});
"

echo "--- İletişim bölümü 2 öğe mi? ---"
echo "$MENU" | node -e "
const d=JSON.parse(require('fs').readFileSync('/dev/stdin','utf8'));
d.forEach(s=>{
  if(s.label && (s.label.toLowerCase().includes('iletişim') || s.label.toLowerCase().includes('iletisim')))
    console.log('İletişim: '+(s.items||[]).length+' öğe '+((s.items||[]).length<=2?'✅':'❌'));
});
"

echo "--- Finans bölümü 4 öğe mi? ---"
echo "$MENU" | node -e "
const d=JSON.parse(require('fs').readFileSync('/dev/stdin','utf8'));
d.forEach(s=>{
  if(s.label && (s.label.toLowerCase().includes('finans')))
    console.log('Finans: '+(s.items||[]).length+' öğe '+((s.items||[]).length<=4?'✅':'❌'));
});
"

echo ""
echo "--- Yatırımcı raporlar görüyor mu? ---"
YAT_MENU=$(curl -s http://localhost:5000/api/me/menu -b /tmp/r2_yatirimci_branch.txt 2>/dev/null)
echo "$YAT_MENU" | grep -i "rapor\|report" && echo "✅ Yatırımcı rapor görüyor" || echo "❌ Yatırımcı rapor görmüyor"

echo ""
echo "--- Fabrika müdür performans görüyor mu? ---"
FM_MENU=$(curl -s http://localhost:5000/api/me/menu -b /tmp/r2_fabrika_mudur.txt 2>/dev/null)
echo "$FM_MENU" | grep -i "performans\|performance" && echo "✅ Fab. müdür performans görüyor" || echo "❌ Fab. müdür performans görmüyor"

echo ""
echo "=== Server durumu ==="
curl -s -o /dev/null -w "HTTP %{http_code}" http://localhost:5000/
echo ""
```
