## FIX 2 — Unified Kiosk Auth Model

### Goal
Both factory and branch kiosks should have the same 2-layer auth:
- **Layer 1:** Kiosk device password (set by admin, same for all tablets at that location)
- **Layer 2:** Personal PIN (per worker)

Branch kiosk already has Layer 1. Factory kiosk doesn't. Add Layer 1 to factory kiosk.

### 2B: Add Device Auth Step to Factory Kiosk Frontend

**File:** `client/src/pages/fabrika/kiosk.tsx`

Add a new first step `device-password` before `select-user`:

```typescript
// Add to steps type:
type KioskStep = 'device-password' | 'select-user' | 'enter-pin' | 'select-station' | 
                  'working' | 'stop-options' | 'production-entry' | 
                  'end-shift-summary' | 'auto-logout' | 'fault-report';

// Initial step:
const [step, setStep] = useState<KioskStep>('device-password');
```

Add device password screen (similar to branch kiosk password screen):

```tsx
{step === 'device-password' && (
  <div className="flex flex-col items-center justify-center min-h-screen bg-background gap-8 px-6">
    {/* DOSPRESSO Logo */}
    <div className="text-center">
      <div className="text-4xl font-black text-foreground mb-2">DOSPRESSO</div>
      <div className="text-sm text-muted-foreground">Fabrika Kiosk Sistemi</div>
    </div>

    {/* Password input */}
    <div className="w-full max-w-sm space-y-4">
      <div className="text-center text-base font-semibold text-foreground">
        Cihaz Şifresi
      </div>
      <input
        type="password"
        value={devicePassword}
        onChange={(e) => setDevicePassword(e.target.value)}
        onKeyDown={(e) => e.key === 'Enter' && handleDeviceAuth()}
        placeholder="Tablet şifresini girin"
        className="w-full text-center text-lg px-4 py-3 rounded-xl border border-border bg-card text-foreground outline-none focus:border-[#122549]"
        autoFocus
      />
      {deviceAuthError && (
        <div className="text-center text-sm text-red-500">{deviceAuthError}</div>
      )}
      <button
        onClick={handleDeviceAuth}
        disabled={!devicePassword || deviceAuthLoading}
        className="w-full py-3 rounded-xl bg-[#122549] text-white font-bold disabled:opacity-40"
      >
        {deviceAuthLoading ? 'Doğrulanıyor...' : 'Giriş'}
      </button>
    </div>
  </div>
)}
```

Add state and handler:
```typescript
const [devicePassword, setDevicePassword] = useState('');
const [deviceAuthError, setDeviceAuthError] = useState('');
const [deviceAuthLoading, setDeviceAuthLoading] = useState(false);

const handleDeviceAuth = async () => {
  if (!devicePassword) return;
  setDeviceAuthLoading(true);
  setDeviceAuthError('');
  try {
    const res = await fetch('/api/factory/kiosk/device-auth', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ devicePassword }),
    });
    if (res.ok) {
      setStep('select-user');
      setDevicePassword(''); // clear for security
    } else {
      setDeviceAuthError('Hatalı şifre. Lütfen tekrar deneyin.');
      setDevicePassword('');
    }
  } catch {
    setDeviceAuthError('Bağlantı hatası');
  } finally {
    setDeviceAuthLoading(false);
  }
};
```
