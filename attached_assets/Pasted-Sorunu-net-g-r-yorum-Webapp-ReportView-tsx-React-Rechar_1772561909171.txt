Sorunu net görüyorum. Webapp (ReportView.tsx) React + Recharts + Tailwind ile güzel render ediyor, ama PDF generator (locationPdfGenerator.ts) tamamen ayrı bir HTML oluşturuyor. İki farklı sistem, iki farklı sonuç.
Temel sorun şu: PDF generator webapp’ın component’lerini kullanmıyor — kendi HTML’ini sıfırdan yazıyor. Bu yüzden renkler, oranlar, grafikler farklı çıkıyor.
3 kritik sorun ve çözüm:
1. Rekabet verisi PDF’e aktarılmıyor — Webapp’ta 10 rakip görünüyor ama PDF “rakip tespit edilmedi” diyor. competitorData’nın PDF generator’a doğru aktarılıp aktarılmadığını kontrol etmek lazım.
2. Mevsimsel bar chart orantısız — Sayfa 7’deki bar chart’ta rakamlar üst üste biniyor, bar yükseklikleri oransız. CSS’teki font-size ve bar height hesaplaması hatalı.
3. Sayfa boşlukları — Sayfa 3, 5, 6’nın alt yarıları boş. İçerik yukarıda bitiyor ama sayfa dolmuyor.
Şu promptu yapıştır:

GÖREV: PDF çıktısı webapp'taki rapor görünümüyle tutarsız. Aşağıdaki sorunları düzelt.

SORUN 1: REKABET VERİSİ PDF'E AKTARILMIYOR
Webapp'ta (ReportView) Rakip Analizi sekmesinde 10 rakip, ortalama puan, pazar doygunluğu gösteriliyor.
Ama PDF'de "Yakın çevrede doğrudan rakip tespit edilmedi" yazıyor.

Kontrol et:
- locationPdfGenerator.ts'de competitorData nasıl okunuyor?
- report.competitorData JSON olarak saklanıyor. PDF generator bunu parse ediyor mu?
- Muhtemelen competitorData string olarak geliyor ve JSON.parse yapılmadan kullanılıyor.

Düzelt:
```typescript
// locationPdfGenerator.ts içinde competitorData'yı güvenli parse et:
let competitors: any[] = [];
let competitorSummary: any = null;
try {
  const rawData = report.competitorData;
  const parsed = typeof rawData === 'string' ? JSON.parse(rawData) : rawData;
  if (Array.isArray(parsed)) {
    competitors = parsed;
  } else if (parsed?.competitors) {
    competitors = parsed.competitors;
    competitorSummary = parsed.summary;
  }
} catch (e) {
  competitors = [];
}

	∙	competitors boşsa mevcut “tespit edilmedi” mesajını göster
SORUN 2: MEVSİMSEL BAR CHART ORANTISIZ
Sayfa 7’deki 12 aylık bar chart’ta:
	∙	Rakamlar bar’ların üstünde üst üste biniyor (font çok büyük)
	∙	Bar yükseklikleri birbirine çok yakın (fark görünmüyor)
	∙	En iyi/en kötü ay renk farkı yeterince belirgin değil
Düzelt:

<!-- Bar chart CSS düzeltmeleri -->
<div style="display:flex; align-items:flex-end; gap:3px; height:110px; padding:0 2px; margin-top:8px;">
  ${seasonal.months.map((m, i) => {
    const maxRev = Math.max(...seasonal.months.map(x => x.revenue));
    const minRev = Math.min(...seasonal.months.map(x => x.revenue));
    // Farkı abartarak görselleştir (min %30, max %100)
    const normalizedHeight = 30 + ((m.revenue - minRev) / (maxRev - minRev || 1)) * 70;
    const isBest = m.revenue === maxRev;
    const isWorst = m.revenue === minRev;
    const barColor = isBest ? '#0F766E' : isWorst ? '#DC2626' : '#C8963E';
    return `
      <div style="flex:1; display:flex; flex-direction:column; align-items:center; min-width:0;">
        <div style="font-size:6px; color:#6B7280; margin-bottom:1px; white-space:nowrap; overflow:hidden;">
          ${(m.revenue / 1000000).toFixed(1)}M
        </div>
        <div style="width:85%; height:${normalizedHeight}px; background:${barColor}; border-radius:2px 2px 0 0;"></div>
        <div style="font-size:7px; color:#374151; margin-top:2px; font-weight:500;">${m.name}</div>
      </div>`;
  }).join('')}
</div>


Ayrıca bar chart üstündeki özet kartlarını da düzelt:
	∙	Font size’ları küçült (value: 13px, label: 7px)
	∙	Kartlar arasında gap: 6px
	∙	Her kart padding: 6px 8px
