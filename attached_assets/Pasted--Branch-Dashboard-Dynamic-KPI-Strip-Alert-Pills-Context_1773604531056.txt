# Branch Dashboard — Dynamic KPI Strip + Alert Pills

## Context
DOSPRESSO franchise platform. Stack: React 18 + TypeScript, TanStack Query v5, Wouter routing.

Problem: dashboard.tsx (branch roles) shows only static module cards.
Three rich branch endpoints already exist with real data — they just aren't wired.

The richest endpoint: GET /api/branch-summary/:branchId
Returns: { branch, kpis: { activeStaff, totalStaff, checklistCompletion, customerAvg, feedbackCount, warnings }, teamStatus[], lowStockItems[], suggestions[] }

Do NOT touch: auth system, HQ dashboards, factory module, DashboardWidgets, CardGridHub.

## TASK 1 — Add KPI + Alert section to dashboard.tsx

**File:** `client/src/pages/dashboard.tsx`

### 1A: Add data fetch for branch users

```tsx
import { useQuery } from '@tanstack/react-query';
import { useAuth } from '@/hooks/use-auth'; // actual path
import { DashboardKpiStrip, type KpiItem } from '@/components/dashboard-kpi-strip';
import { DashboardAlertPills, type AlertPill } from '@/components/dashboard-alert-pills';
import { useMemo } from 'react';

// Inside the branch section of the component, add:
const { user } = useAuth();

const { data: branchSummary } = useQuery({
  queryKey: ['/api/branch-summary', user?.branchId],
  queryFn: () => fetch(`/api/branch-summary/${user?.branchId}`).then(r => r.json()),
  enabled: !!user?.branchId,
  staleTime: 3 * 60 * 1000,    // 3 minutes
  refetchInterval: 5 * 60 * 1000, // auto-refresh every 5 min
});
```

### 1B: Build KPI items from real data

```tsx
const kpiItems = useMemo((): KpiItem[] => {
  const kpis = branchSummary?.kpis;
  return [
    {
      value: kpis?.activeStaff?.toString() ?? '—',
      label: 'Aktif Personel',
      color: undefined,
    },
    {
      value: kpis?.checklistCompletion != null
        ? `%${Math.round(kpis.checklistCompletion)}`
        : '—',
      label: 'Checklist',
      color: kpis?.checklistCompletion != null && kpis.checklistCompletion < 70
        ? '#dc2626'
        : undefined,
    },
    {
      value: kpis?.customerAvg != null
        ? `${kpis.customerAvg.toFixed(1)}/5`
        : '—',
      label: 'Müşteri Puanı',
      color: kpis?.customerAvg != null && kpis.customerAvg < 3.5
        ? '#f97316'
        : '#16a34a',
    },
    {
      value: kpis?.warnings?.toString() ?? '0',
      label: 'Uyarı',
      color: (kpis?.warnings ?? 0) > 0 ? '#dc2626' : undefined,
    },
  ];
}, [branchSummary]);
```

### 1C: Build alert pills from real data

```tsx
const alertPills = useMemo((): AlertPill[] => {
  const pills: AlertPill[] = [];
  const kpis = branchSummary?.kpis;

  if ((kpis?.warnings ?? 0) > 0) {
    pills.push({ label: `${kpis.warnings} uyarı`, variant: 'red', dot: true });
  }

  if (kpis?.checklistCompletion != null && kpis.checklistCompletion < 70) {
    pills.push({ label: `Checklist %${Math.round(kpis.checklistCompletion)}`, variant: 'orange', dot: true });
  }

  if ((branchSummary?.lowStockItems?.length ?? 0) > 0) {
    pills.push({ label: `${branchSummary.lowStockItems.length} düşük stok`, variant: 'orange', dot: true });
  }

  if (kpis?.customerAvg != null && kpis.customerAvg >= 4.0) {
    pills.push({ label: `Müşteri ${kpis.customerAvg.toFixed(1)}/5 ⭐`, variant: 'green', dot: true });
  }

  if (pills.length === 0) {
    pills.push({ label: 'Şube normal', variant: 'green', dot: true });
  }

  return pills;
}, [branchSummary]);
```
