## Sprint 1 — Bölüm A: DB Hazırlık

### A1: support_tickets tablosuna eksik kolonlar ekle (⚠ DANIŞMA)

```sql
-- Önce mevcut yapıyı kontrol et, sonra eksik olanları ekle
-- ticket_type: kanal belirleme
ALTER TABLE support_tickets ADD COLUMN IF NOT EXISTS ticket_type VARCHAR(50) DEFAULT 'franchise_talep';
-- rating: müşteri puanı (Misafir kanalı için)
ALTER TABLE support_tickets ADD COLUMN IF NOT EXISTS rating INTEGER;
-- rating_hizmet, rating_temizlik, rating_urun, rating_personel: detay puanlar
ALTER TABLE support_tickets ADD COLUMN IF NOT EXISTS rating_hizmet INTEGER;
ALTER TABLE support_tickets ADD COLUMN IF NOT EXISTS rating_temizlik INTEGER;
ALTER TABLE support_tickets ADD COLUMN IF NOT EXISTS rating_urun INTEGER;
ALTER TABLE support_tickets ADD COLUMN IF NOT EXISTS rating_personel INTEGER;
-- source: nereden geldi (qr, web, app, manual, escalation)
ALTER TABLE support_tickets ADD COLUMN IF NOT EXISTS source VARCHAR(30) DEFAULT 'manual';
-- channel: misafir | franchise
ALTER TABLE support_tickets ADD COLUMN IF NOT EXISTS channel VARCHAR(20) DEFAULT 'franchise';
```
