## YAPILAN DEĞİŞİKLİKLER — IT Danışman Ne Düzeltti?

### Değişiklik 1: Chunk Crash Fix (KRİTİK)
**Dosya:** `client/src/pages/akademi-v3/index.tsx`
**Neden:** 4 tab (HomeTab, TrainingTab, WebinarTab, CareerTab) `lazy()` ile
yükleniyordu. Deploy/rebuild sonrası eski chunk hash'leri değişince tarayıcı
yeni chunk'ı bulamıyordu → "Importing a module script failed" hatası.
**Düzeltme:** `lazy()` → `lazyWithRetry()` değiştirildi. Bu fonksiyon 3 kere
retry yapıyor, başarısız olursa sayfayı yeniliyor. App.tsx'teki tüm sayfalar
zaten bunu kullanıyordu — akademi tab'ları eksikti.
```
ESKİ: const HomeTab = lazy(() => import("./HomeTab"));
YENİ: const HomeTab = lazyWithRetry(() => import("./HomeTab"));
```

### Değişiklik 2: HQ Rol Tanıma Genişletme (KRİTİK)
**Dosya:** `client/src/pages/akademi-v3/index.tsx`
**Neden:** Akademi sayfası HQ rollerini hardcoded 7 elemanlı bir array ile
kontrol ediyordu: `['admin','ceo','cgo','coach','trainer','kalite_kontrol','gida_muhendisi']`.
Bu listede `recete_gm`, `sef`, `fabrika_mudur`, `muhasebe_ik`, `satinalma`,
`marketing` gibi roller YOKTU. Bu roller akademi sayfasına girince HQ yönetim
paneline yönlendirilmek yerine öğrenci görünümünü görüyordu.
**Düzeltme:** Hardcoded array kaldırıldı, yerine `isHQRole()` fonksiyonu
(shared/schema'dan import) kullanıldı. Bu fonksiyon tüm HQ rollerini otomatik
tanır — yeni rol eklendiğinde de çalışır.
```
ESKİ: const HQ_ROLES = ['admin','ceo','cgo','coach','trainer','kalite_kontrol','gida_muhendisi'];
      const isHQUser = user && HQ_ROLES.includes(user.role);
YENİ: const isHQUser = user && isHQRole(user.role);
```

### Değişiklik 3: Akademi Sidebar Path Düzeltme
**Dosya:** `client/src/components/layout/module-menu-config.ts`
**Neden:** Akademi sidebar'ında 3 item yanlış path kullanıyordu:
- "Eğitim Modülleri" → `/egitim` (bu path hard redirect yapıyor, tam sayfa yenileniyor)
- "Webinar" → `/akademi` (tab parametresi yok, ana sayfaya düşüyor)
- "Kariyer Yolu" → `/akademi` (aynı sorun)
**Düzeltme:** Query parameter ile doğru tab'a yönlendirme eklendi:
```
ESKİ: { path: "/egitim" }     → hard redirect, sayfa yenileniyor
YENİ: { path: "/akademi?tab=egitim" }  → SPA navigasyonu, doğru tab açılıyor

ESKİ: { path: "/akademi" }    → ana sayfa, webinar tab değil
YENİ: { path: "/akademi?tab=webinar" } → webinar tab açılıyor

ESKİ: { path: "/akademi" }    → ana sayfa, kariyer tab değil
YENİ: { path: "/akademi?tab=kariyer" } → kariyer tab açılıyor
```
