### T002 — Add pre-build duplicate variable check script

Create `scripts/check-duplicates.sh`:
```bash
#!/bin/bash
# Check for duplicate isLoading/refetch destructuring in page files
ERRORS=0
for f in $(find client/src/pages -name "*.tsx"); do
  DUPES=$(grep -c "isLoading }" "$f" 2>/dev/null)
  if [ "$DUPES" -gt "1" ]; then
    echo "❌ DUPLICATE isLoading in $f"
    ERRORS=$((ERRORS+1))
  fi
  DUPES2=$(grep -c "refetch }" "$f" 2>/dev/null)
  if [ "$DUPES2" -gt "1" ]; then
    echo "❌ DUPLICATE refetch in $f"
    ERRORS=$((ERRORS+1))
  fi
done
if [ "$ERRORS" -gt "0" ]; then
  echo "Found $ERRORS duplicate variable issues. Fix before deploying."
  exit 1
fi
echo "✅ No duplicates found"
```

Add to package.json scripts:
```json
"check": "bash scripts/check-duplicates.sh && vite build"
```
